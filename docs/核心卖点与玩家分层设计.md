## 一、设计总纲

### 1.1 核心设计理念

**玩家分层，各取所需，各有价值**

- 题材玩家 → 用时间换内容，提供活跃底座
- 玩法玩家 → 用钱换时间，提供稳定收入
- 鲸鱼玩家 → 用钱换稀缺，提供深度付费和竞争氛围

### 1.2 核心卖点一句话

**"盗墓不是取宝，是补天——五层真相层层反转，三层克制策略为王"**

## 二、玩家三层模型

### 2.1 模型总览

```
┌───────────────────────────────────────────┐
│  外层：题材玩家（活跃底座）                │
│  - 盗墓爱好者                              │
│  - 追求：剧情、角色、氛围                  │
│  - 价值：DAU、口碑、社区热度               │
│  - 付费：不要求                            │
│  - 行为：用时间换内容                      │
├───────────────────────────────────────────┤
│  中层：玩法玩家（付费主力）                │
│  - 手游玩家                                │
│  - 追求：角色收集、养成进度                │
│  - 价值：稳定付费、中长期留存              │
│  - 付费：抽卡、养成资源                    │
│  - 行为：用钱换时间                        │
├───────────────────────────────────────────┤
│  内层：鲸鱼玩家（深度付费）                │
│  - 重度氪金玩家                            │
│  - 追求：超稀有角色、满养成、PVP顶端       │
│  - 价值：高ARPU、排行榜竞争氛围            │
│  - 付费：限定角色、满养成、PVP特权         │
│  - 行为：用钱换稀缺                        │
└───────────────────────────────────────────┘
```

### 2.2 各层玩家画像

#### 题材玩家（外层）

| 维度 | 内容 |
|------|------|
| **人口占比** | 50-60% |
| **来源** | 盗墓小说粉丝、悬疑爱好者、传统文化爱好者 |
| **核心诉求** | 好看的剧情、喜欢的角色、沉浸的氛围 |
| **游玩节奏** | 轻度，每日30分钟以内 |
| **付费意愿** | 低，可能买外观 |
| **流失风险** | 剧情看完就走 |
| **留存关键** | 持续更新剧情、角色个人故事 |

#### 玩法玩家（中层）

| 维度 | 内容 |
|------|------|
| **人口占比** | 30-40% |
| **来源** | 手游玩家、卡牌玩家、养成玩家 |
| **核心诉求** | 角色收集、养成进度、有策略但不肝 |
| **游玩节奏** | 中度，每日1-2小时 |
| **付费意愿** | 中等，月卡+赛季通行证 |
| **流失风险** | 养成卡顿、角色抽不到、PVP被碾压 |
| **留存关键** | 稳定养成节奏、角色获取友好、PVP体验 |

#### 鲸鱼玩家（内层）

| 维度 | 内容 |
|------|------|
| **人口占比** | 5-10% |
| **来源** | 重度手游玩家、竞技玩家、收藏玩家 |
| **核心诉求** | 稀缺性、差异化、成就感 |
| **游玩节奏** | 重度，每日2小时+ |
| **付费意愿** | 高，全内容购买 |
| **流失风险** | 缺乏竞争、内容消耗完 |
| **留存关键** | 稀有角色、排行榜、GVG地位 |

### 2.3 各层诉求对比

| 维度 | 题材层 | 玩法层 | 鲸鱼层 |
|------|--------|--------|--------|
| **剧情** | 核心，必须精彩 | 有就行，可跳过 | 不在意 |
| **角色** | 喜欢就行，不追求全 | 要抽到，慢慢养 | 要全，要满破 |
| **养成** | 不深挖，跟着剧情走 | 有进度感就行 | 要深度，要差异化 |
| **PVP** | 不关心 | 偶尔玩玩 | 要冲榜，要优势 |
| **GVG** | 参与即贡献 | 主力输出 | 精英攻坚 |
| **稀有度** | 不在意 | 有SSR就开心 | 要UR/限定，要稀有 |
| **外观** | 可能买 | 有更好 | 必须有专属 |

## 三、各层核心卖点

### 3.1 题材层卖点

**核心口号："盗墓不是取宝，是补天"**

| 卖点 | 具体内容 | 竞品对比 |
|------|---------|---------|
| **世界观独特性** | 古墓即星盘，棺椁即星位，机关即星轨 | 市场无同类题材 |
| **五层真相** | 层层反转，最终三结局 | 手游中罕见的深度叙事 |
| **角色故事** | 养成角色解锁TA的人生 | 角色不只是数值，有灵魂 |
| **传承体系** | 四大传承与真实历史关联 | 有文化厚度 |

**体验设计要点**：
- 剧情解锁与养成进度绑定，驱动活跃
- 每个角色都有个人剧情线
- 剧情演出质量要高（立绘、配音、音乐）
- 轻度玩家也能轻松体验主线

### 3.2 玩法层卖点

**核心口号："三层克制，SR也能打UR"**

| 卖点 | 具体内容 | 竞品对比 |
|------|---------|---------|
| **策略友好** | 克制系统放大策略，缩小数值差距 | 比纯数值游戏友好 |
| **收集深度** | 四大传承×五行属性×多种定位 | 角色体系有深度 |
| **养成节奏稳定** | 1个月1个5星，3个月3个6星 | 可预期，不焦虑 |
| **PVP体验保护** | 分段匹配+奖励分层 | 不会被碾压 |

**体验设计要点**：
- SSR不是必需，SR也能用
- 养成周期明确，不肝不氪
- PVP分段匹配，同段位竞争
- 奖励以外观为主，不绑数值

### 3.3 鲸鱼层卖点

**核心口号："限定角色，全服唯一"**

| 卖点 | 具体内容 | 竞品对比 |
|------|---------|---------|
| **稀缺性** | UR限定角色，赛季专属 | 真正稀缺，不是换皮 |
| **差异化** | 专属皮肤、特效、称号 | 别人没有的，我有 |
| **成就感** | 排行榜、GVG地位、全角色收藏 | 社交货币充足 |
| **加速体验** | 付费快速养成，先人一步 | 时间vs金钱选择 |

**体验设计要点**：
- UR真正稀缺，不是所有人都有
- 专属外观要足够炫
- 排行榜曝光度高
- GVG中是公会核心战力

## 四、PVP与GVG系统设计

### 4.1 双轨设计理念

| 玩法 | 定位 | 参与门槛 | 鲸鱼优势 |
|------|------|---------|---------|
| **1v1 PVP** | 个人竞技 | 30级解锁 | 30%以内（分段匹配保护） |
| **GVG公会战** | 团体对抗 | 加入公会即可 | 1个鲸鱼≈5个中氪，但人数可制衡 |

### 4.2 1v1 PVP设计

**核心机制**：
- 全自动战斗（已确定）
- 分段匹配：青铜打青铜，钻石打钻石
- 克制保护：策略克制放大，有机会弥补数值差距
- 奖励分层：各段位奖励相近，外观为主

**分段体系**：

| 段位 | 分数区间 | 人口比例 | 奖励特点 |
|------|---------|---------|---------|
| 青铜 | 0-999 | 20% | 基础奖励 |
| 白银 | 1000-1999 | 30% | 基础+皮肤碎片 |
| 黄金 | 2000-2999 | 25% | 皮肤+SSR碎片 |
| 铂金 | 3000-3999 | 15% | 限定皮肤+SSR保底 |
| 钻石 | 4000-4999 | 7% | 专属皮肤+UR碎片 |
| 大师 | 5000-5999 | 2.5% | 专属皮肤+UR保底 |
| 宗师 | 6000+ | 0.5% | 全套专属+限定角色 |

**保护机制**：
- 分段匹配：同段位±1段位内匹配
- 胜率平衡：连败保护，匹配略弱对手
- 时间限制：匹配超时30秒，扩大范围

### 4.3 GVG公会战设计

**核心逻辑**：
```
鲸鱼玩家数量有限
    ↓
一个公会鲸鱼再多也有限
    ↓
公会需要人数制衡
    ↓
人越多越强
    ↓
题材玩家也有存在意义
```

**战力换算**：
- 1个鲸鱼玩家 ≈ 5个玩法层玩家
- 但10个玩法层玩家 > 1个鲸鱼玩家
- 公会战是人数×战力的综合

**GVG玩法设计**：

| 阶段 | 时长 | 内容 |
|------|------|------|
| 报名期 | 1天 | 公会报名，分配对手 |
| 准备期 | 1天 | 策略布置，分配据点 |
| 战斗期 | 2天 | 攻防战，消耗体力 |
| 结算期 | 1天 | 奖励发放，排名公布 |

**据点类型**：

| 据点 | 防守难度 | 攻打要求 | 战略价值 |
|------|---------|---------|---------|
| 核心据点 | 高 | 需要鲸鱼攻坚 | 决定胜负 |
| 中型据点 | 中 | 玩法层主力 | 次要目标 |
| 小型据点 | 低 | 题材层可打 | 消耗战 |

**各层玩家GVG定位**：

| 玩家层 | GVG角色 | 贡献方式 | 获得奖励 |
|--------|--------|---------|---------|
| **题材层** | 人数 | 参与即贡献，消耗对方体力 | 公会基础奖励、归属感 |
| **玩法层** | 主力 | 稳定输出，攻占中型据点 | 公会奖励、排名奖励 |
| **鲸鱼层** | 精英 | 攻坚核心据点、防守关键位置 | 排行榜、专属称号、公会地位 |

**GVG奖励设计**：

| 奖励类型 | 获取条件 | 是否影响数值 |
|---------|---------|-------------|
| 公会币 | 参与即可获得 | ✅ 可兑换养成资源 |
| 公会皮肤 | 公会排名奖励 | ❌ 纯外观 |
| 专属称号 | 公会前10名 | ❌ 社交货币 |
| SSR碎片 | 个人贡献前50% | ✅ 养成加速 |
| UR碎片 | 个人贡献前10% | ✅ 限定获取 |

### 4.4 鲸鱼优势与平衡

**鲸鱼优势来源**：

| 维度 | 具体优势 | 上限控制 |
|------|---------|---------|
| 角色强度 | UR比SSR强15% | 数值差距小 |
| 养成速度 | 1个月满养成 vs 3个月 | 时间换金钱 |
| PVP段位 | 更快上分 | 分段匹配 |
| GVG贡献 | 1人打5人 | 人数可制衡 |
| 外观 | 专属皮肤、特效 | 纯展示 |

**平衡设计要点**：
- 数值差距控制在30%以内
- 策略克制可以弥补差距
- GVG人数制衡，不让鲸鱼一人打全公会
- 排行榜分多个维度（不只是战力）

## 五、商业化策略

### 5.1 各层付费设计

#### 题材层（不要求付费）

| 付费点 | 设计 |
|--------|------|
| 外观 | 角色/皮肤，纯展示 |
| 剧情解锁 | 支线故事付费解锁 |
| 去广告 | 一次性小额付费 |

**关键**：不卡进度，不强制付费

#### 玩法层（稳定付费）

| 付费点 | 价格 | 价值 |
|--------|------|------|
| 月卡 | 30元/月 | 每日资源+特权，养成加速 |
| 赛季通行证 | 68元/赛季 | 赛季专属奖励 |
| 抽卡 | 6元/抽 | 角色获取 |
| 养成资源 | 小额 | 加速养成 |

**关键**：性价比高，稳定投入有回报

#### 鲸鱼层（深度付费）

| 付费点 | 价格 | 价值 |
|--------|------|------|
| 限定角色 | 直购或高概率池 | 稀缺性 |
| 满养成加速 | 中额 | 先人一步 |
| 专属外观 | 中额 | 差异化 |
| 排行榜特权 | 长期付费 | 成就感 |

**关键**：真正稀缺，物有所值

### 5.2 收入结构预期

| 玩家层 | 人口占比 | 收入占比 | ARPU |
|--------|---------|---------|------|
| 题材层 | 50-60% | 5-10% | 低 |
| 玩法层 | 30-40% | 40-50% | 中 |
| 鲸鱼层 | 5-10% | 40-50% | 高 |

**关键**：玩法层是收入主力，鲸鱼层是ARPU主力

## 六、留存设计

### 6.1 各层留存关键

#### 题材层留存

| 留存点 | 具体设计 |
|--------|---------|
| **剧情更新** | 每2-3个月一个大版本，新剧情 |
| **角色故事** | 每个角色都有个人剧情线 |
| **日常活跃** | 养成角色解锁新剧情 |
| **社区氛围** | 剧情/角色讨论热度 |

**流失风险**：剧情看完就走
**应对**：持续更新剧情，角色个人故事

#### 玩法层留存

| 留存点 | 具体设计 |
|--------|---------|
| **养成进度** | 稳定可预期，不焦虑 |
| **角色收集** | 新角色持续更新 |
| **PVP体验** | 分段匹配，不被碾压 |
| **GVG参与** | 公会归属感，有贡献 |

**流失风险**：养成卡顿、抽不到角色、PVP被碾压
**应对**：养成节奏稳定、角色获取友好、PVP保护

#### 鲸鱼层留存

| 留存点 | 具体设计 |
|--------|---------|
| **稀缺内容** | 限定角色、专属外观 |
| **竞争氛围** | 排行榜、GVG地位 |
| **内容消耗** | 深度养成、全角色收藏 |
| **社交货币** | 称号、展示框、排行榜 |

**流失风险**：缺乏竞争、内容消耗完
**应对**：持续更新限定内容、GVG竞争、排行榜多维度

### 6.2 日常活跃设计

| 活跃行为 | 题材层 | 玩法层 | 鲸鱼层 |
|---------|--------|--------|--------|
| 每日任务 | ✓ 轻松完成 | ✓ 完整完成 | ✓ 快速完成 |
| 剧情推进 | 核心动力 | 随缘 | 不在意 |
| 角色养成 | 跟着剧情走 | 有节奏推进 | 快速满养成 |
| PVP | 不参与 | 偶尔参与 | 核心内容 |
| GVG | 参与 | 主力 | 精英 |

## 七、总结

### 7.1 核心卖点矩阵

| 卖点 | 题材层 | 玩法层 | 鲸鱼层 |
|------|--------|--------|--------|
| 世界观独特性 | ★★★★★ | ★★★ | ★★ |
| 五层真相剧情 | ★★★★★ | ★★ | ★ |
| 角色故事 | ★★★★★ | ★★★ | ★★ |
| 三层克制策略 | ★★ | ★★★★★ | ★★★★ |
| 角色收集深度 | ★★★ | ★★★★★ | ★★★★★ |
| 养成节奏稳定 | ★★ | ★★★★★ | ★★★ |
| PVP竞技 | ★ | ★★★ | ★★★★★ |
| GVG公会战 | ★★★ | ★★★★ | ★★★★★ |
| 限定角色稀缺 | ★ | ★★★ | ★★★★★ |
| 专属外观 | ★★★ | ★★★ | ★★★★★ |

### 7.2 设计核心原则

1. **题材玩家不卡进度**——轻松体验剧情
2. **玩法玩家不被碾压**——分段匹配+策略克制
3. **鲸鱼玩家有价值**——稀缺内容+竞争氛围
4. **人数制衡氪金**——GVG设计让每个玩家都有用
5. **持续更新内容**——剧情+角色+活动
