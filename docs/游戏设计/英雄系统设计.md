## 一、系统概述

### 1.1 设计理念

**核心定位**：英雄系统是本游戏的核心收集养成内容，玩家通过获取和培养英雄，组建探险队伍，挑战各种古墓关卡。

**设计原则**：
- **多英雄友好**：单个英雄养成维度精简，避免单英雄养成过重
- **传承特色鲜明**：四大传承各有特色，玩家可按喜好选择
- **品质差异合理**：高品质英雄稀有但非必须，低品质英雄也有价值
- **获取途径多元**：抽卡、契约、关卡奖励、活动等多种获取方式

### 1.2 英雄总数与分布

**上线时英雄数量：40个**

| 品质 | 数量 | 占比 | 定位 |
|:----:|:----:|:----:|------|
| UR | 1 | 2.5% | 最稀有，限定获取，顶级强度 |
| SSR | 4 | 10% | 超稀有，高培养价值，阵容核心 |
| SR | 10 | 25% | 稀有，主流阵容核心，稳定输出 |
| R | 15 | 37.5% | 普通，容易获取，中前期主力 |
| N | 10 | 25% | 常见，初期过渡使用，可突破提升 |

**四大传承分布**：每传承10个英雄

| 传承 | 数量 | 特色定位 |
|------|:----:|----------|
| 星纹摸金 | 10 | 暴击、穿透、爆发、定位寻宝 |
| 璇玑搬山 | 10 | 控制、机关、AOE、阵法破解 |
| 辰宿卸岭 | 10 | 生存、护盾、反击、力量对抗 |
| 天枢发丘 | 10 | 治疗、增益、复活、Boss弱点 |

## 二、四大传承体系

### 2.1 传承设计理念

**传承是英雄的核心身份标识**，决定英雄的：
- 战斗风格和定位
- 特殊能力（传承技能）
- 适合的机关类型
- 阵容搭配方向

### 2.2 星纹摸金

**传承定位**：输出型传承，擅长暴击与穿透

| 维度 | 设计 |
|------|------|
| 核心机制 | 星纹印记、暴击加成、穿透无视防御 |
| 战斗风格 | 高爆发、快速击杀、单体突破 |
| 适合玩家 | 追求高爆发、快速通关的玩家 |
| 传承技能 | 星纹探测（探查隐藏奖励）、摸金之力（暴击伤害提升） |
| 代表英雄 | 摸金校尉(SSR)、摸金手(SR)、追星者(SR) |

**英雄定位分布**：
- 远程输出：摸金手、星箭、追星者
- 刺客/切后：小星、黑衣人、穿山甲
- 爆发核心：摸金校尉
- 辅助：星尘
- 过渡：铁蛋

### 2.3 璇玑搬山

**传承定位**：控制型传承，擅长机关破解与AOE

| 维度 | 设计 |
|------|------|
| 核心机制 | 璇玑阵法、群体控制、机关识破 |
| 战斗风格 | 控场、群体伤害、机关克星 |
| 适合玩家 | 策略型玩家，喜欢控场和破解机关 |
| 传承技能 | 搬山卸岭（范围伤害）、机关识破（解除陷阱） |
| 代表英雄 | 璇玑子(SSR)、星祭司(SSR)、机关师(SR) |

**英雄定位分布**：
- 法师：星祭司、璇玑子
- 控制：搬山客、阵法师、灵蛊师
- 输出：机关师、破阵者、傀儡师
- 辅助：小璇、小云

### 2.4 辰宿卸岭

**传承定位**：防御型传承，擅长生存与反击

| 维度 | 设计 |
|------|------|
| 核心机制 | 土系护盾、生命回复、反击反伤 |
| 战斗风格 | 持久战、护盾保护、反击输出 |
| 适合玩家 | 稳扎稳打、持久战玩家 |
| 传承技能 | 卸岭甲胄（护盾加成）、大地之力（生命提升） |
| 代表英雄 | 卸岭力士(SR)、守陵人(SR)、盾甲(SR) |

**英雄定位分布**：
- 坦克：铁柱、岩甲、土行孙、盾甲、卸岭力士
- 反击：守陵人
- 战士：阿强、蛮牛、地龙

### 2.5 天枢发丘

**传承定位**：辅助型传承，擅长治疗与增益

| 维度 | 设计 |
|------|------|
| 核心机制 | 天枢指针、Boss弱点、复活重生 |
| 战斗风格 | 团队辅助、治疗复活、Boss战专家 |
| 适合玩家 | 团队配合、辅助型玩家 |
| 传承技能 | 发丘指印（弱点标记）、天命回春（复活队友） |
| 代表英雄 | 发丘天官(SSR)、神医(SR)、灵儿(R) |

**英雄定位分布**：
- 治疗：神医、灵儿、小月
- 辅助：发丘天官、引路人
- 增益：天枢使者、祈福者
- 复活：渡魂人

## 三、英雄品质系统

### 3.1 品质差异设计

**品质影响**：
- 基础属性（高品质属性更高）
- 技能强度（高品质技能更强）
- 成长上限（高品质成长空间更大）

**品质不影响**：
- 技能数量（所有英雄统一3技能）
- 养成深度（所有英雄养成路径相同）
- 使用价值（低品质英雄也有独特价值）

### 3.2 各品质定位

| 品质 | 定位 | 获取难度 | 养成建议 |
|------|------|---------|---------|
| UR | 追求目标 | 极难 | 长期培养，顶级阵容核心 |
| SSR | 核心主力 | 难 | 重点培养，阵容核心 |
| SR | 主力选择 | 中等 | 稳定培养，中后期主力 |
| R | 过渡主力 | 容易 | 适度培养，前期主力 |
| N | 初期过渡 | 非常容易 | 初期使用，可突破提升 |

### 3.3 N品质英雄的突破设计

**设计理念**：N品质英雄虽然基础较弱，但可通过突破系统提升至更高品质。

**突破路线**：
- N → R（突破1次）
- R → SR（突破2次）
- SR → SSR（突破3次）

**突破价值**：
- 让N品质英雄有长期价值
- 免费玩家可通过突破获得高品质英雄
- 增加养成深度和目标感

## 四、英雄定位与职责

### 4.1 定位分类

| 定位 | 职责 | 特点 | 代表英雄 |
|------|------|------|---------|
| 坦克 | 承受伤害、保护队友 | 高HP、高防御、护盾技能 | 铁柱、盾甲、卸岭力士 |
| 战士 | 前排输出、承伤兼顾 | 平衡型、持续输出 | 阿强、蛮牛、地龙 |
| 输出 | 主要伤害来源 | 高攻击、爆发技能 | 摸金手、机关师、破阵者 |
| 刺客 | 切后排、击杀脆皮 | 高速度、高爆发、低防御 | 小星、黑衣人、穿山甲 |
| 法师 | 群体伤害、控制 | AOE技能、控制能力 | 星祭司、璇玑子 |
| 控制 | 限制敌人行动 | 眩晕、定身、减速 | 搬山客、阵法师、灵蛊师 |
| 辅助 | 增益队友、提供效果 | 增益、护盾、怒气 | 星尘、小璇、小云 |
| 治疗 | 恢复队友生命 | 治疗技能、复活能力 | 神医、灵儿、小月 |

### 4.2 阵容搭配原则

**推荐阵容结构**：
- 坦克 1-2人
- 输出 2-3人
- 辅助/治疗 1-2人

**传承搭配**：
- 纯传承阵容：同一传承英雄，传承加成最大
- 混合传承：不同传承互补，应对各种场景
- 机关对应：根据关卡机关类型选择对应传承

## 五、英雄技能系统

### 5.1 技能结构

**统一3技能设计**：所有英雄固定3个技能

| 技能类型 | 说明 | 使用方式 |
|----------|------|----------|
| 普攻 | 基础攻击技能 | 无消耗，每回合可用 |
| 怒攻 | 消耗怒气的强力技能 | 消耗怒气，高伤害/特殊效果 |
| 被动 | 自动生效的效果 | 无需操作，持续生效 |

### 5.2 技能差异化

**品质差异体现在技能强度**，而非技能数量：

| 品质 | 普攻系数 | 怒攻系数 | 被动效果 |
|:----:|:--------:|:--------:|----------|
| N | 80-95% | 180-220% | 单一效果 |
| R | 100% | 200-250% | 效果增强 |
| SR | 100-110% | 250-320% | 双重效果 |
| SSR | 115% | 400-500% | 强力效果 |
| UR | 120%+ | 600%+ | 独特机制 |

### 5.3 技能随突破提升

技能不单独养成，随突破自动提升：

| 突破阶段 | 技能提升 |
|----------|----------|
| 1阶 | 普攻等级+1 |
| 2阶 | 被动效果+20% |
| 3阶 | 怒攻等级+2 |
| 4阶 | 全技能等级+3 |

## 六、英雄获取方式

### 6.1 获取途径汇总

| 获取方式 | 英雄品质 | 说明 |
|----------|---------|------|
| 新手引导 | N/R | 铁柱、小璇等初期英雄 |
| 关卡奖励 | N/R/SR | 通关章节获得固定英雄 |
| 招募抽卡 | R/SR/SSR/UR | 主要获取方式 |
| 契约任务 | SR/SSR | 完成英雄契约加入 |
| 首充礼包 | SR/SSR | 首充奖励英雄 |
| 限时活动 | SR/SSR/UR | 活动限定英雄 |
| 商店兑换 | R/SR | 积分兑换英雄碎片 |

### 6.2 前期英雄获取节奏

| 顺序 | 英雄 | 获取方式 | 解锁时机 |
|:----:|------|---------|---------|
| 1 | 玩家 | 开场 | - |
| 2 | 陈星垣 | 营地加入 | 第1关 |
| 3 | 铁柱 | 契约任务 | 第2关 |
| 4 | 云绮罗 | 新手招募池 | 第3关 |

### 6.3 新手招募池设计

**第3关解锁招募系统后**：
- 云绮罗保底（10抽必得）
- 费率折扣（比普通池便宜）
- 限定次数（新手专属，过期作废）

## 七、英雄养成系统

### 7.1 养成维度

| 维度 | 说明 | 是否可转移 |
|------|------|:----------:|
| 等级 | 消耗经验药水提升 | ❌ |
| 突破 | 解锁等级上限，提升技能 | ❌ |
| 星级 | 消耗碎片升星 | ❌ |
| 宝物 | 装备盗墓宝物 | ✅ |

### 7.2 养成节奏

| 阶段 | 养成重点 | 时间投入 |
|------|---------|---------|
| 初期(1-30级) | 等级提升 | 1-3天 |
| 前期(30-70级) | 第一次突破 | 1周 |
| 中期(70-120级) | 突破+升星 | 1-2月 |
| 后期(120级+) | 深度养成 | 长期 |

## 八、英雄与战斗系统

### 8.1 英雄在战斗中的定位

- **出战位置**：最多6人上阵（随等级解锁）
- **战斗控制**：半自动战斗，怒攻手动释放
- **传承加成**：同传承英雄有额外加成

### 8.2 英雄与机关的互动

| 传承 | 擅长机关 | 互动方式 |
|------|---------|---------|
| 星纹摸金 | 星盘、方位 | 快速定位、破解星象 |
| 璇玑搬山 | 阵法、机关 | 识破机关、快速破解 |
| 辰宿卸岭 | 力量型障碍 | 推开石门、承受机关 |
| 天枢发丘 | 封印、感知 | 解除封印、感知危险 |

## 九、英雄与探险队系统

### 9.1 英雄占用机制

**派出探险的英雄不能同时参与主线关卡**

**策略决策**：
- 派谁去探险？（收益导向）
- 留谁在队伍？（战斗导向）
- 高战力英雄的资源分配

### 9.2 英雄类型与探险任务

| 任务类型 | 推荐英雄类型 | 加成 |
|----------|-------------|:----:|
| 资源探索 | 任意 | +0% |
| 碎片搜寻 | 对应传承英雄 | +15% |
| 宝物鉴定 | 探测型英雄 | +15% |
| 守护挑战 | 坦克型英雄 | +15% |
| 传承试炼 | 对应传承英雄 | +20% |

## 十、设计总结

### 10.1 系统特色

1. **传承特色鲜明**：四大传承各有定位，玩家可按喜好选择
2. **品质差异合理**：低品质英雄也有价值，可通过突破提升
3. **获取途径多元**：抽卡、契约、关卡奖励等多种方式
4. **养成深度适中**：单个英雄养成维度精简，多英雄养成友好

### 10.2 后续扩展

- 新英雄持续更新
- 英雄时装系统
- 英雄羁绊系统
- 英雄觉醒系统
