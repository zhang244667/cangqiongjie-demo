## 一、英雄立绘生成规范

### 1.1 关键词标准格式

```
纯白背景游戏角色立绘，[角色描述]，写实国漫风格，暗黑奇幻，完整全身立绘，四肢完整，不裁切，纯白色背景
```

### 1.2 格式拆解

| 部分 | 内容 | 说明 |
|------|------|------|
| 背景声明 | 纯白背景游戏角色立绘 | 开头声明，便于后期抠图 |
| 角色描述 | [性别][体型][服装][道具][气质] | 核心视觉元素 |
| 风格标签 | 写实国漫风格，暗黑奇幻 | 统一美术风格 |
| 完整性 | 完整全身立绘，四肢完整，不裁切 | 防止边缘裁切 |
| 背景结尾 | 纯白色背景 | 再次强调背景 |

**男性角色示例**：

**壮汉角色示例**：

| 标签 | 适用场景 |

## 二、立绘后期处理流程

### 2.1 工具准备

使用 Python + rembg 库进行批量抠图：

```bash
pip install rembg
```

### 2.2 单张抠图脚本

```python
from rembg import remove

input_path = "输入图片路径.jpg"
output_path = "输出图片路径.png"

with open(input_path, 'rb') as f:
    input_data = f.read()

output_data = remove(input_data)

with open(output_path, 'wb') as f:
    f.write(output_data)
```

### 2.3 批量抠图脚本

```python
from rembg import remove
import os

input_dir = "输入文件夹路径"
output_dir = "输出文件夹路径"

for filename in os.listdir(input_dir):
    if filename.endswith(('.jpg', '.jpeg', '.png')):
        input_path = os.path.join(input_dir, filename)
        output_path = os.path.join(output_dir, filename.replace('.jpg', '.png'))
        
        with open(input_path, 'rb') as f:
            output_data = remove(f.read())
        
        with open(output_path, 'wb') as f:
            f.write(output_data)
        
        print(f"完成: {filename}")
```

## 六、注意事项

1. **背景必须纯白**：便于rembg自动抠图
2. **风格标签统一**：所有角色使用"写实国漫风格，暗黑奇幻"
3. **完整性关键词**：必须包含"完整全身立绘，四肢完整，不裁切"
4. **尺寸固定**：1600×2848，无需额外指定
5. **批量处理**：生成多张后再统一抠图，提高效率

## 七、常见问题与解决方案

### 7.1 Q版/卡通化问题

**问题表现**：生成的角色变成Q版比例（头身比1:3），过于可爱化

**原因**：使用了"娇小"、"俏皮"、"可爱"等词汇

**错误关键词示例**：
```
身材娇小的年轻刺客，机灵俏皮的表情
```

**正确关键词示例**：
```
年轻刺客少年，敏捷身姿，眼神灵动
```

**解决方案**：
- ❌ 避免：娇小、俏皮、可爱、萌、Q版
- ✅ 使用：年轻、敏捷、灵动、矫健

### 7.2 边缘裁切问题

**问题表现**：手部、脚部、武器边缘被裁切

**解决方案**：必须包含完整性关键词
```
完整全身立绘，四肢完整，不裁切
```

### 7.3 风格不一致问题

**问题表现**：不同角色风格差异大（有的3D，有的二次元，有的写实）

**解决方案**：统一使用风格标签
```
写实国漫风格，暗黑奇幻
```

### 7.4 大体型角色裁切问题

**问题表现**：体型大+武器大的角色（如两米巨汉手持重锤）边缘被裁切

**错误尝试**：使用"适度缩小"会导致角色比例被压缩，变矮壮

**解决方案**：使用构图控制词，不改变角色比例
```
构图收紧边距充足
```

**正确示例**：
```
纯白背景游戏角色立绘，身高两米高大威猛巨汉，身穿粗布劲装，手持巨大星纹重锤，写实国漫风格，暗黑奇幻，完整全身立绘，四肢完整，不裁切，构图收紧边距充足，纯白色背景
```

**关键原则**：
- ❌ "适度缩小" → 角色比例被压缩
- ✓ "构图收紧边距充足" → 角色比例不变，画面留白增加

### 7.5 浅色武器抠图缺失问题

**问题表现**：rembg抠图后，浅色/金属光泽武器部分被误删，呈现半透明虚影

**原因**：浅色武器与纯白背景颜色相近，rembg误判为背景

**解决方案**：使用**深灰色背景**替代纯白背景

**正确示例**：
```
深灰色背景游戏角色立绘，[角色描述]，写实国漫风格，暗黑奇幻，完整全身立绘，四肢完整，不裁切，构图收紧边距充足，深灰色背景
```

**背景选择对照表**：

| 角色/武器特征 | 推荐背景 | 原因 |
|--------------|---------|------|
| 深色服装+深色武器 | 纯白色 | 对比度充足 |
| 深色服装+浅色武器 | 深灰色 | 避免浅色武器被误删 |
| 浅色服装+任意武器 | 深灰色 | 避免服装细节丢失 |

### 7.6 武器动作导致裁切问题

**问题表现**：武器在动作姿态下超出画面边缘被裁切（如拉弓动作导致弓箭横向超出）

**原因**：动作姿态让武器向画面边缘延伸，9:16竖构图空间有限

**错误示例**：
```
手持星纹宝弓（AI会生成为拉弓动作）
```

**解决方案**：
1. 改为**静态站立姿态**
2. 武器改为**背负**或**收起状态**
3. 手持其他道具（如摸金符）

**正确示例**：
```
背负星纹宝弓，手持摸金符，静态站立姿态
```

**武器姿态对照表**：

| 武器类型 | ❌ 动作姿态（易裁切） | ✓ 静态姿态 |
|---------|---------------------|-----------|
| 弓箭 | 手持弓做拉弓动作 | 背负弓，手持其他道具 |
| 长枪 | 双手持枪前刺 | 单手持枪竖立，或背负 |
| 双刀 | 双手各持一刀展开 | 单刀佩腰，单刀手持 |
| 重锤 | 双手举锤挥击 | 单手扶锤站立 |

### 7.7 纯白背景抠图残留问题

**问题表现**：rembg抠图后背景有残留，没有完全抠干净

**原因**：AI生成的"纯白背景"并非绝对纯白，可能有柔和光影/渐变，导致与角色边缘难以区分

**解决方案**：改用深灰色背景生成，再抠图

**对比效果**：

| 背景 | 抠图效果 | 适用场景 |
|------|---------|---------|
| 纯白 | 可能有残留 | 背景完全无光影时可用 |
| 深灰 | 干净彻底 | 推荐，通用解决方案 |

**注意**：抠图效果和角色形象可能存在权衡，用户可根据实际需求选择版本

### 7.8 年轻角色二次元化问题

**问题表现**：年轻男性/女性角色生成为二次元动漫风格

**原因**：使用"年轻"、"眼神专注"等抽象描述，AI容易生成动漫形象

**错误示例**：
```
身材精瘦的年轻男性，眼神专注
```

**解决方案**：
1. 用具体年龄替代"年轻"（如"二十五岁成年男性"）
2. 用具体面部特征替代抽象描述
3. 添加成熟气质关键词

**正确示例**：
```
二十五岁成年男性探险者，面容沉稳，干练气质
```

**词汇对照表**：

| ❌ 易二次元化 | ✓ 写实风格 |
|--------------|-----------|
| 年轻男性/女性 | XX岁成年男性/女性 |
| 眼神专注/坚毅 | 面容沉稳/成熟 |
| 精瘦身材 | 结实/健壮身材 |

### 7.9 浅色辅助元素抠图问题

**问题表现**：角色身旁的浅色元素（傀儡、宠物、召唤物）被rembg误删或变模糊

**原因**：浅色元素与深灰背景对比不足

**解决方案**：使用**黑色背景** + 提示词强调辅助元素的深色版本

**对比效果**：

| 背景 | 适用场景 |
|------|---------|
| 纯白 | 单一角色，深色服装 |
| 深灰 | 多数情况通用 |
| 黑色 | 有浅色辅助元素（傀儡/宠物/召唤物） |

**正确示例**：
```
黑色背景游戏角色立绘，术士手指缠满丝线，身旁有深色机关傀儡，黑色背景
```

### 7.10 复杂角色抠图问题汇总

**问题1：大型武器与身体穿插**
- 案例：卸岭力士"扛着巨大石柱"
- 原因：武器穿过身体部位
- 解决：用"单手扶持竖立"让武器在身体侧边

**问题2：浅色部位被误删**
- 案例：地龙尾巴、蛮牛武器
- 原因：浅色部位与背景对比不足
- 解决：在提示词中强调"深色XX" + 使用黑色背景

**问题3：浅色特效丢失**
- 案例：地龙毒雾
- 原因：浅色特效（烟雾、光芒）与背景相近
- 解决：黑色背景 + 强调"亮色特效清晰可见"

**综合解决方案对照表**：

| 角色特征 | 推荐背景 | 提示词要点 |
|---------|---------|-----------|
| 普通角色 | 深灰 | 标准描述 |
| 浅色服装 | 黑色 | 强调"深色版本" |
| 浅色武器 | 黑色 | 强调"深色武器" |
| 浅色特效（烟雾/光芒） | 黑色 | 强调"亮色特效清晰可见" |
| 浅色尾巴/肢体 | 黑色 | 强调"深色尾部" |
| 大型武器 | 深灰/黑色 | 用"扶持竖立"避免穿插 |

**关键原则**：
- 背景越深，抠图越干净，但需要角色/武器/特效都是深色
- 有浅色元素时，要么改背景，要么在提示词中强调该元素是深色
- **纯黑背景必须加"无渐变"**：避免AI生成带光影的黑色背景导致抠图残留

**问题4：黑色背景抠图残留**
- 案例：蛮牛v2头盔上方、腿部中间残留黑色
- 原因：AI生成的"黑色背景"带有光影渐变，不是纯黑，rembg无法完全识别
- 解决：提示词加"纯黑背景无渐变"

**正确写法对比**：
```
❌ 黑色背景 → AI生成带光影的黑色背景 → 抠图残留
✓ 纯黑背景无渐变 → AI生成纯净黑色 → 抠图干净
```

## 九、角色多姿态生成方法

### 9.1 基本原理

使用已验证的立绘作为**参考图**，配合简洁的动作描述，生成同一角色的不同姿态版本（战斗/剧情等）。

### 9.2 操作步骤

**第一步**：获取原图URL
```python
# 使用file_to_url工具
file_to_url(file_path="imgs/原图路径.jpg")
# 返回URL: https://www.coze.cn/s/xxx/
```

**第二步**：生成新姿态
```
prompt: "简洁的动作描述"
ref_images: ["原图URL"]
mode: "生图"
```

### 9.3 实际案例

**原图**：摸金手静态站立立绘

**战斗姿态生成**：
- prompt: "手持星纹弩箭射击姿态，战斗动作"
- ref_images: ["https://www.coze.cn/s/ei9TujbTacU/"]
- mode: "生图"

**生成效果**：
- ✅ 探险风格服饰保持一致
- ✅ 角色体型气质基本一致
- ⚠️ 面部细节有差异（发型、五官会变化）
- ✅ 动作姿态可控

### 9.4 注意事项

1. **prompt要简洁**：只需描述动作，不需要重复角色描述
2. **一致性有限**：角色整体风格一致，但面部细节会有差异
3. **适用场景**：战斗姿态、剧情动作、不同表情等
4. **后处理**：生成后可人工筛选最接近的版本

### 9.5 适用场景对照表

| 场景 | 推荐做法 |
|------|---------|
| 战斗姿态 | 参考图 + "射击姿态/挥剑姿态" |
| 剧情动作 | 参考图 + "坐姿/行走姿态" |
| 不同表情 | 参考图 + "微笑/严肃表情" |
| 不同服装 | 参考图 + "穿铠甲/穿礼服" |
