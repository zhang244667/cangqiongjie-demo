# 《苍穹劫·摸金传人》角色设定文档

## 一、主要角色设定

### 1.1 主角：陈星垣

#### 基础信息

| 项目 | 内容 |
|------|------|
| **姓名** | 陈星垣 |
| **性别** | 男 |
| **年龄** | 24岁（外表）/ 实际承受诅咒年龄35岁 |
| **身份** | 末代星纹摸金传人 |
| **传承** | 星纹摸金（主）+ 四脉兼修（成长后解锁） |

#### 外貌特征

- **眉心星纹**：淡紫色六芒星印记，觉醒后逐渐变亮
- **眼神**：清冷深邃，偶有星点光芒闪烁
- **发色**：墨黑色，常年略显凌乱
- **肤色**：苍白，有长期诅咒消耗的痕迹
- **着装**：黑色探险服，内衬绣有星纹的道袍，腰间系着爷爷留下的罗盘
- **特殊**：左手腕有家族符文刺青，是封印印记

#### 性格特点

| 性格维度 | 表现 |
|----------|------|
| **核心** | 隐忍、执着、重情重义 |
| **优点** | 冷静判断、顾全大局、善于学习 |
| **缺点** | 不善表达、习惯独自承担、容易自责 |
| **成长** | 从逃避命运到接受使命，从独狼到团队领袖 |

#### 能力设定

**传承技能**：
- 观星定穴（感知龙脉走向）
- 分金定穴（确定棺椁位置）
- 星文解读（解读古墓铭文）
- 星纹觉醒（激活后可短暂预见未来）

**属性成长**：
- 攻击成长：S级
- 生命成长：A级
- 速度成长：A级
- 防御成长：B级

#### 背景故事

陈星垣出生于星纹摸金一脉的最后传人家庭。家族世代被苍穹劫诅咒，无一人活过三十岁。

**童年**：
- 六岁时第一次目睹父亲因诅咒发作死去
- 爷爷陈玄机开始亲自教导他盗墓技艺
- 发现他眉心有异常强烈的星纹

**少年**：
- 十八岁时爷爷预言他活不过三十岁
- 拒绝接受命运，开始独自研究破解诅咒的方法
- 爷爷失踪后，踏上寻找真相的道路

**成年**：
- 二十四岁时爷爷突然失踪
- 收到爷爷留下的《穹墓纪要》和半张星图
- 正式踏入盗墓者的世界

#### 核心动机

1. **表层动机**：寻找失踪的爷爷
2. **中层动机**：破解家族诅咒
3. **深层动机**：了解真相，做出选择
4. **终极动机**：掌握自己的命运

#### 台词风格

| 场景 | 台词风格 | 示例 |
|------|----------|------|
| 日常 | 简洁、略带疏离 | *"嗯。""不用管我。""走吧。"* |
| 队友互动 | 温和、偶尔调侃 | *"你比我爷爷还啰嗦。""小心点，别踩机关。"* |
| 战斗 | 冷静、果断 | *"我来开路。""掩护我。""撤退！"* |
| 诅咒发作 | 压抑、隐忍 | *"......没事。继续走。""我还能撑住。"* |
| 真相揭示 | 震惊、挣扎 | *"这不可能......爷爷他......"* |
| 坚定决心 | 低沉、有力 | *"不管真相是什么，我都要亲眼看到。"* |

### 1.2 队友角色

#### 1.2.1 云绮罗

| 项目 | 内容 |
|------|------|
| **姓名** | 云绮罗 |
| **性别** | 女 |
| **年龄** | 22岁 |
| **身份** | 璇玑搬山传人 |
| **传承** | 璇玑搬山 |

**外貌特征**：
- 长发及腰，常用发簪束起
- 眼眸为淡青色，如深潭古井
- 肤若凝脂，身着改良汉服
- 手持星盘法器

**性格特点**：
- 外表温婉，内心坚韧
- 擅长星术，但不愿轻易使用
- 有自己的秘密和使命
- 对陈星垣有特殊的情感

**能力设定**：
- 观星术（预知危险）
- 避劫符（驱散负面状态）
- 陨星傀儡（召唤傀儡助战）
- 回春术（治疗队友）

**背景故事**：
璇玑搬山一脉世代承受另一种诅咒——每代传人都会被星力反噬，容貌会逐渐"星化"，最终化为星尘消散。云绮罗的母亲正是如此消失的。

她加入探险队的目的不仅是寻找解除诅咒的方法，更是要找到当年导致搬山一脉被诅咒的真正原因。

**台词风格**：
> *"星垣，前面有危险......我看到了。"*
> *"别逞强了，让我给你画个护身符。"*
> *"我的母亲......也是这样消失的。"*

> *"归墟那帮孙子......迟早有一天我要灭了他们。"*

他加入探险队一方面是为了赎罪，一方面是弟弟临终前让他保护好卸岭一脉最后的传人（陈星垣）。

## 二、四大传承角色群像

### 2.1 星纹摸金一脉

**家族成员**：
| 世代 | 姓名 | 结局 | 备注 |
|------|------|------|------|
| 1 | 陈玄甲 | 寿终正寝 | 开创星纹一脉 |

> *"星落之处，吾之所在；命数已定，吾之不屈。"*

**特殊能力**：星纹觉醒后可看见常人看不见的东西——龙脉星气、封印裂痕、命运丝线

---

### 2.2 天枢发丘一脉

**组织结构**：
```
发丘天师（1人）
    ↓
发丘校尉（3人）
    ↓
发丘力士（若干）
```

**传承信物**：发丘天星印（已失落多年）

**传承信条**：
> *"一印镇千邪，一刀破万阵。"*

**组织现状**：隐世家族，传承断断续续，萧破军是目前唯一活跃的传人

---

### 2.3 璇玑搬山一脉

**诅咒来源**：搬山先祖曾试图挖开某座禁忌之墓，惊动了沉睡的上古存在，被降下诅咒

**诅咒表现**：
- 每代传人都会被星力"星化"
- 最终化为星尘消散
- 寿命不定，平均30-40岁

**传承信条**：
> *"星陨如泪，吾心如磐；纵化尘埃，亦护苍生。"*

**解除方法**：未知（云绮罗正在寻找）

---

### 2.4 辰宿卸岭一脉

**传承方式**：师徒传承为主，血脉传承为辅

**分支**：
- 官派卸岭（曾为朝廷服务）
- 草莽卸岭（江湖人士）
- 石敢当一脉（矿工出身）

**传承信条**：
> *"山高人为峰，岭深吾敢当。"*

**特殊之处**：与守墓神族有血缘关系，世代对立

## 四、角色对话风格指南

### 4.1 主角对话风格

**总体风格**：简洁、沉稳、偶尔带有一丝自嘲

| 情绪状态 | 说话特点 | 示例 |
|----------|----------|------|
| 平静 | 简短回应，不多说废话 | *"嗯。" / "走吧。" / "知道了。"* |
| 关心队友 | 话不多但暖 | *"别逞强。" / "我帮你。"* |
| 独处时 | 内心独白较多 | *"爷爷......你到底想让我怎么做？"* |
| 诅咒发作 | 压抑、隐忍 | *"......没事，继续。"* |
| 愤怒 | 少见但可怕 | *"......你们不该碰这里的东西。"* |
| 决断 | 坚定有力 | *"这是我的选择。"* |

### 4.2 云绮罗对话风格

**总体风格**：温婉、含蓄、偶尔俏皮

| 场景 | 说话特点 | 示例 |
|------|----------|------|
| 日常 | 温柔体贴 | *"星垣，早饭要记得吃。"* |
| 战斗 | 冷静专业 | *"前方三米有机关，我来处理。"* |
| 调侃 | 难得活泼 | *"陈大传人，连路都走不稳？"* |
| 关心 | 委婉表达 | *"今晚星象不好......要不要我陪你？"* |
| 悲伤 | 隐忍 | *"母亲她......也是这样的夜晚离开的。"* |

### 4.3 萧破军对话风格

**总体风格**：豪爽、直接、江湖气

| 场景 | 说话特点 | 示例 |
|------|----------|------|
| 日常 | 大大咧咧 | *"今天天气不错！适合下斗！"* |
| 战斗 | 热血激励 | *"弟兄们，冲！让这帮粽子知道厉害！"* |
| 生气 | 暴躁直接 | *"他妈的！老子今天非弄死你不可！"* |
| 严肃 | 认真郑重 | *"星垣，这件事我必须跟你说清楚。"* |

### 4.4 石敢当对话风格

**总体风格**：沉默寡言、惜字如金、行动派

| 场景 | 说话特点 | 示例 |
