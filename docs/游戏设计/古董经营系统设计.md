# 《苍穹劫·摸金传人》古董经营系统设计文档

> 版本：1.0
> 更新日期：2026年4月24日
> 设计定位：模拟经营+挂机收益

## 一、系统概述

### 1.1 设计理念

**核心定位**：古董经营系统是本游戏的挂机收益核心，融合模拟经营玩法，玩家通过经营古董店获得稳定的离线收益。

**核心体验**：
- 下墓获取古董，放入店铺自动售卖
- 升级店铺装修，提升全局属性
- 拍卖会信息博弈，高端古董交易

### 1.2 宝物与古董的区分

```
┌─────────────────────────────────────────────────────────────┐
│                    宝物 vs 古董                              │
├──────────────────────┬──────────────────────────────────────┤
│        宝物          │               古董                   │
├──────────────────────┼──────────────────────────────────────┤
│ 黑驴蹄子、摸金符等   │ 瓷器、鼎、玉器等收藏品               │
│ 英雄装备             │ 古董店商品                           │
│ 提升英雄属性         │ 售卖获得金币                         │
│ 多余可做材料         │ 珍品可放珍宝阁                       │
└──────────────────────┴──────────────────────────────────────┘
```

### 1.3 解锁条件

**第五关后开放古董经营系统**

开放原因：
- 玩家已有4-5个英雄，全局属性加成有意义
- 玩家有养成基础，需要新的目标
- 英雄数量足够支持多队伍探险

解锁后玩家获得：
- 古董店基础功能
- 拍卖会参与资格
- 珍宝阁入口

### 1.4 系统关联

```
古墓探险 ─────→ 获取古董 ─────→ 古董店售卖 ──→ 金币
    │                               │              │
    │                               ↓              ↓
    │                         拍卖会交易      升级店铺装修
    │                               │              │
    └───────────────────────────────┴──────────────┘
                                         ↓
                                   全局属性提升
```

## 二、古董店系统

### 2.1 核心循环

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   下墓获取古董 → 放入店铺 → NPC自动购买 → 获得金币         │
│        ↓                                     ↓              │
│   古董分级                              升级装修            │
│        ↓                                     ↓              │
│   售价差异                              全局属性            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 古董分级

| 级别 | 产出来源 | 售价区间 | 说明 |
|------|---------|---------|------|
| **普通** | 前期古墓（1-3关） | 100-500金币 | 常见古董，大量产出 |
| **稀有** | 中期古墓（4-6关） | 500-2000金币 | 较为少见，有一定价值 |
| **精品** | 后期古墓（7关+） | 2000-10000金币 | 稀有古董，高价值 |
| **珍品** | Boss/隐藏关卡 | 10000-50000金币 | 极稀有，建议拍卖会交易 |

**卖出速度**：固定速度，不因古董级别而变化

### 2.3 店铺装修系统

#### 装修区域

| 区域 | 装修内容 | 属性方向 | 说明 |
|------|---------|---------|------|
| **展示区** | 展柜、展台、底座 | 全局攻击、生命、防御 | 核心展示区域 |
| **灯光区** | 吊灯、射灯、氛围灯 | 探险效率、战斗暴击 | 提升观赏效果 |
| **门面区** | 招牌、橱窗、门头 | NPC好感度、顾客吸引 | 门面担当 |
| **仓储区** | 仓库、货架、保险柜 | 库存容量、古董保鲜 | 扩大经营规模 |
| **鉴定区** | 鉴定台、放大镜、工具 | 鉴定速度、鉴宝能力 | 专业鉴定空间 |

#### 装修等级

每个区域可独立升级，共10级：
- 等级越高，属性加成越强
- 升级消耗金币，费用递增
- 满级解锁特殊效果

```
示例：展示区升级

等级1：木质展柜 → 全局攻击+1%
等级2：玻璃展柜 → 全局攻击+2%
等级3：红木展柜 → 全局攻击+3%
...
等级10：紫檀雕花展柜 → 全局攻击+10%，解锁"展柜光环"效果
```

### 2.4 顾客系统

#### NPC顾客行为

| 顾客类型 | 特征 | 购买偏好 | 行为模式 |
|---------|------|---------|---------|
| **普通顾客** | 随机出现 | 任意古董 | 快速购买，不挑剔 |
| **收藏家** | 较少出现 | 稀有/精品 | 愿出高价，停留时间长 |
| **土豪** | 罕见出现 | 不差钱 | 一次购买多件，溢价购买 |
| **专家** | 特定时间出现 | 特定类型 | 眼光毒辣，只买真品 |

#### 购买逻辑

- NPC顾客自动上门，自动购买
- 玩家不参与定价，售价按古董级别固定
- 顾客类型影响购买频率和数量
- 离线期间持续产生收益

### 2.5 离线收益

#### 收益规则

**免费玩家**：
- 古董库存量少，可能出现空窗期
- 空窗期无收益，需上线补充库存

**付费玩家**：
- 古董库存量大，24小时持续有货
- 离线收益稳定，无空窗期

#### 收益结算

- 上线时一次性结算离线收益
- 显示离线时长和收益明细
- 最多计算24小时离线收益

## 三、拍卖会系统

### 3.1 系统定位

| 对比项 | 古董店 | 拍卖会 |
|-------|-------|-------|
| **定位** | 日常销售 | 高端交易 |
| **价格** | 固定（按级别） | 浮动（信息博弈） |
| **古董类型** | 普通/稀有/精品 | 精品/珍品 |
| **参与方式** | 自动售卖 | 主动参与 |

### 3.2 拍卖会玩法

#### 玩家角色

玩家可以扮演三种角色：

| 角色 | 行为 | 收益 |
|------|------|------|
| **买家** | 竞拍古董 | 获得古董（低价捡漏） |
| **卖家** | 委托拍卖 | 获得金币（高价出手） |
| **观察者** | 观看拍卖 | 获取信息（了解市场） |

#### 信息分层

| 信息层级 | 内容 | 获取方式 |
|---------|------|---------|
| **公开信息** | 古董基础描述、起拍价 | 免费获取 |
| **隐藏信息** | 真伪、真实价值、稀有度 | 需要投入资源获取 |
| **误导信息** | 假情报、托儿出价 | 需要玩家判断 |

#### 信息获取途径

拍卖前信息搜集阶段：

| 信息来源 | 可信度 | 消耗资源 | 风险 |
|---------|:------:|---------|------|
| **古董商** | 中等 | 金币 | 可能提供假信息 |
| **竞拍者** | 低 | 社交互动 | 可能故意误导 |
| **观摩者** | 中高 | 金币/时间 | 相对可靠 |
| **专家鉴定** | 高 | 鉴宝令 | 最可靠但最贵 |

### 3.3 NPC竞拍者设计

| 类型 | 出价风格 | 信息可信度 | 特征描述 |
|------|---------|:----------:|---------|
| **土豪** | 豪爽不差钱 | 低 | 出价快，溢价高，经常看走眼 |
| **精明人** | 锱铢必较 | 高 | 出价慢，会放弃，眼光毒 |
| **托儿** | 故意抬价 | 假信息 | 和某个卖家串通 |
| **行家** | 只收精品 | 极高 | 遇到好货才出手 |
| **新手** | 犹豫不决 | 中等 | 容易被其他人影响 |

### 3.4 拍卖会流程

```
┌─────────────────────────────────────────────────────────────┐
│                      拍卖会流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   阶段1：信息搜集（5分钟）                                  │
│   ├── 查看古董基础信息                                     │
│   ├── 花费资源获取隐藏信息                                 │
│   ├── 与NPC交流获取情报                                    │
│   └── 判断古董真伪和价值                                   │
│                                                             │
│   阶段2：竞价拍卖（实时）                                   │
│   ├── 多人轮流出价                                         │
│   ├── 观察他人出价行为                                     │
│   ├── 决策是否继续跟进                                     │
│   └── 成交或流拍                                           │
│                                                             │
│   阶段3：结算                                               │
│   ├── 买家支付金币，获得古董                               │
│   ├── 卖家获得成交价（扣除手续费）                         │
│   └── 显示古董真实价值（验证判断）                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.5 风险设计

**买家风险**：
- 买到赝品：古董价值远低于成交价
- 信息误导：被托儿带节奏，溢价购买
- 流拍损失：预付保证金不退

**卖家风险**：
- 流拍：古董砸手里，浪费拍卖位
- 低价成交：信息不足，真品被低价买走
- 手续费：无论成交与否，都需要支付手续费

### 3.6 拍卖会触发

| 触发方式 | 频率 | 古董品质 | 说明 |
|---------|------|---------|------|
| **自动刷新** | 每12小时 | 随机 | 系统自动生成拍卖会 |
| **手动触发** | 消耗鉴宝令（每日3次） | 随机 | 玩家主动触发 |
| **主线触发** | 通关大关 | 高品质 | 特殊拍卖会，稀有古董 |

## 四、珍宝阁系统

### 4.1 系统定位

珍宝阁是玩家存放珍品古董的场所，放入珍品可获得特殊效果。

### 4.2 珍宝阁功能

| 功能 | 说明 | 效果 |
|------|------|------|
| **展示收藏** | 展示玩家获得的珍品古董 | 成就展示 |
| **特殊加成** | 珍品放入后获得额外效果 | 小范围属性加成 |
| **解锁内容** | 收集特定古董组合 | 解锁隐藏内容 |

### 4.3 珍品获取

- 主要途径：拍卖会竞拍
- 次要途径：Boss关卡掉落
- 特殊途径：隐藏关卡/活动

## 六、与其他系统的关联

### 6.1 与主线关卡的关联

```
主线进度 → 解锁古墓类型 → 影响古董产出 → 影响古董店收益
```

### 6.2 与探险队系统的关联

```
探险队 → 古墓探险 → 获取古董 → 补充古董店库存
```

### 6.3 与英雄系统的关联

```
店铺装修 → 全局属性 → 所有英雄受益
珍宝阁 → 特殊加成 → 特定英雄受益
```

### 6.4 与经济系统的关联

```
古董店收益 → 金币 → 英雄养成/店铺装修
拍卖会 → 大额金币流动 → 经济调节
```

## 七、设计总结

### 7.1 系统特色

1. **模拟经营+挂机**：古董店自动售卖，离线持续收益
2. **信息博弈**：拍卖会核心玩法，判断真伪、洞察人心
3. **养成深度**：店铺装修、珍宝阁收藏，长期养成目标
4. **付费友好**：免费玩家可正常体验，付费玩家体验更流畅

### 7.2 后续可扩展

- 古董鉴定小游戏
- 玩家间古董交易
- 古董收藏成就系统
- 特殊古董剧情解锁
