> 版本：1.0
> 更新日期：2026年4月24日

---

## 一、设计理念

### 1.1 核心原则

**关卡驱动**：系统解锁以主线关卡进度为主要依据，而非时间天数
**渐进开放**：新玩家不会一次性面对过多系统，逐步深入
**需求匹配**：系统在玩家需要时解锁，而非提前或滞后
**体验节奏**：保证每1-2关有新内容解锁，维持新鲜感

## 二、系统开放详细配置

### 2.1 第一关（入门阶段）

**关卡定位**：玩家初次体验，建立基础认知

| 系统/功能 | 解锁时机 | 说明 |
|----------|---------|------|
| 基础战斗 | 开场 | 战斗教学、技能释放 |
| 火把视野 | 开场 | 核心探索机制 |
| 基础机关 | 开场 | 机关解谜入门 |
| 陈星垣加入 | 第一关结束回营地 | 第一个英雄加入队伍 |
| 英雄养成 | 第一关结束回营地 | 等级提升 |
| 编队系统 | 第一关结束回营地 | 队伍配置（2人） |

**新手引导流程**：
```
第一关结束引导：
1. 回到营地
2. 陈星垣加入队伍（剧情）
3. 引导打开英雄界面
4. 教学英雄升级（消耗经验药水）
5. 完成养成教学
```

**养成系统说明**：
- 等级：消耗经验药水提升
- 技能：随突破自动提升，无需单独养成
- 宝物：第4关解锁

**玩家体验目标**：
- 理解战斗机制
- 获得第一个英雄
- 学会英雄升级

---

### 2.2 第二关（扩展阶段）

**关卡定位**：玩法扩展，引入社交元素

| 系统/功能 | 解锁时机 | 说明 |
|----------|---------|------|
| 铁柱契约任务 | 第二关结束回营地 | 引导体验竞技场 |
| 竞技场系统 | 第二关结束 | PVP玩法解锁 |
| 编队扩展 | 竞技场教学后 | 队伍配置（3人） |
| 竞技场挑战券 | 竞技场解锁时 | 每日挑战次数 |

**契约任务：铁柱加入**
```
铁柱契约任务流程：
1. 第二关结束回营地
2. 铁柱出现，表示想见识玩家实力
3. 引导前往竞技场
4. 完成3次竞技场挑战
5. 铁柱契约完成，加入队伍
```

**玩家体验目标**：
- 体验契约英雄获取
- 解锁竞技场玩法
- 队伍扩展到3人

---

### 2.3 第三关（扩展阶段）

**关卡定位**：英雄扩充，抽卡引导

| 系统/功能 | 解锁时机 | 说明 |
|----------|---------|------|
| 招募系统 | 第三关结束 | 抽卡解锁，引导获取新英雄 |
| 新手招募池 | 第三关结束 | 新手专属卡池，云绮罗保底 |

**上阵位置解锁（随等级）**：

| 玩家等级 | 上阵位置 | 说明 |
|:--------:|:--------:|------|
| 1级 | 2人 | 初始 |
| 5级 | 3人 | 第一个扩展位置 |
| 10级 | 4人 | 第二个扩展位置 |
| 20级 | 5人 | 第三个扩展位置 |
| 30级 | 6人 | 满编 |

**引导流程**：
```
第三关结束引导：
1. 系统提示"招募系统已解锁"
2. 引导前往招募系统
3. 展示新手招募池（云绮罗保底）
4. 引导抽卡获得云绮罗
5. 云绮罗编入队伍
```

**新手招募池设计**：
- 云绮罗保底（10抽必得）
- 费率折扣（比普通池便宜）
- 限定次数（新手专属，过期作废）

**玩家体验目标**：
- 解锁招募系统
- 通过新手池获得云绮罗
- 等级提升解锁更多上阵位置

---

### 2.4 第四关（养成深化）

**关卡定位**：英雄养成深化

| 系统/功能 | 解锁时机 | 说明 |
|----------|---------|------|
| 宝物系统 | 第四关结束 | 英雄装备宝物 |

**宝物系统解锁内容**：
- 宝物装备栏（每英雄1槽）
- 宝物强化
- 宝物合成

**玩家体验目标**：
- 英雄养成深化
- 宝物收集与强化

---

### 2.5 第五关（经营阶段）

**关卡定位**：挂机收益，模拟经营

| 系统/功能 | 解锁时机 | 说明 |
|----------|---------|------|
| 古董店系统 | 第五关结束 | 模拟经营+挂机收益 |
| 拍卖会系统 | 第五关结束 | 信息博弈+高端交易 |
| 珍宝阁系统 | 第五关结束 | 珍品收藏+特殊加成 |
| 探险队系统 | 第五关结束 | 核心挂机玩法 |

**开放原因**：
- 玩家已有4-5个英雄，探险队可派多队
- 全局属性加成对多个英雄有意义
- 玩家有养成基础，需要新的目标

**玩家体验目标**：
- 获得稳定挂机收益
- 体验拍卖会信息博弈
- 开始古董收藏
- 多队伍探险

---

### 2.6 第六关及之后（深度阶段）

**关卡定位**：玩法深度挖掘

| 系统/功能 | 解锁时机 | 说明 |
|----------|---------|------|
| 竞技场深度 | 第六关+ | 排位赛、赛季奖励 |
| 探险队扩展 | 第六关+ | 更多队伍、更高难度 |
| 高级副本 | 第七关+ | 组队副本、挑战副本 |
| 公会系统 | 第九关+ | 社交玩法（待定） |
| 英雄觉醒 | 第十一关+ | 英雄突破系统 |

**玩家体验目标**：
- 深度养成
- 社交玩法
- 挑战更高难度

## 六、玩家行为节奏

### 6.1 活跃玩家每日行为（第3关后）

```
┌─────────────────────────────────────────────────────────────┐
│                    活跃玩家每日行为                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  上午（上线10分钟）                                         │
│  ├── 领取离线收益（古董店）                                │
│  ├── 派出探险队（4队）                                     │
│  ├── 收取探险队回报（昨日派出）                            │
│  └── 清理日常任务                                          │
│                                                             │
│  中午（上线20分钟）                                         │
│  ├── 主线推图（1-2关）                                     │
│  ├── 竞技场挑战（10-30次）                                 │
│  └── 英雄养成（升级、装备）                                │
│                                                             │
│  晚上（上线30分钟）                                         │
│  ├── 拍卖会参与（如有刷新）                                │
│  ├── 古董店管理（补货、装修）                              │
│  ├── 高级副本（如有解锁）                                  │
│  └── 社交（公会、聊天）                                    │
│                                                             │
│  离线期间                                                   │
│  ├── 古董店自动售卖                                        │
│  └── 探险队自动执行                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 玩家分层行为差异

| 行为 | 免费玩家 | 月卡玩家 | 鲸鱼玩家 |
|------|---------|---------|---------|
| 每日上线次数 | 2-3次 | 3-4次 | 4-5次 |
| 每日时长 | 30-60分钟 | 60-90分钟 | 90-120分钟 |
| 竞技场挑战 | 10次 | 20-30次 | 50次+ |
| 探险队效率 | 基础 | +20% | +50% |
| 古董店收益 | 有空窗期 | 稳定 | 24h满载 |
