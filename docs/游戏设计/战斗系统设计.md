# 战斗系统设计文档

## 一、核心机制概述

### 战斗流程
```
【行动条】按速度值排序显示行动顺序
    ↓
【自动普攻】轮到英雄时自动攻击目标，积攒怒气
    ↓
【技能插队】玩家点击怒气满的英雄 → 立即释放大招
    ↓
【敌人行动】轮到敌人时自动攻击/蓄力/召唤
    ↓
循环直到战斗结束
```

### 核心特点
- **半自动战斗**：普攻自动进行，技能由玩家主动释放
- **行动条系统**：按速度值决定行动顺序
- **插队机制**：怒气满时可随时点击释放技能，不打断行动条
- **独立血条**：每个英雄独立HP，死亡后无法行动

## 二、英雄属性系统

### 属性定义

| 属性 | 说明 | 影响 |
|------|------|------|
| HP | 生命值 | 降到0则英雄死亡 |
| 攻击 | 攻击力 | 影响普攻和技能伤害 |
| 防御 | 防御力 | 减少受到的伤害 |
| 速度 | 行动速度 | 决定行动条顺序 |

### 速度计算
```
最终速度 = 职业基础速度 + 装备加成 + 技能加成 + Buff/Debuff

行动间隔 = 1000 / 最终速度（游戏帧）
```

| 速度值 | 行动频率 |
|--------|----------|
| 85 | 每11.8帧行动1次 |
| 95 | 每10.5帧行动1次 |
| 100（基准） | 每10帧行动1次 |
| 105 | 每9.5帧行动1次 |
| 110 | 每9.1帧行动1次 |

## 四、怒气系统

### 怒气获取

| 来源 | 怒气值 |
|------|--------|
| 普通攻击 | +20 |
| 受到伤害 | +5 |
| 击杀敌人 | +15 |

### 怒气消耗

| 行为 | 怒气消耗 |
|------|----------|
| 释放大招 | 100 |

### 怒气节奏
- 普攻5次可放大招（约2-3回合）
- 受击可加速怒气积累
- 放完大招怒气归零

## 五、战斗伤害计算

### 伤害公式
```
基础伤害 = 攻击力 - 防御力
实际伤害 = 基础伤害 × 技能倍率 × 暴击倍率 × 各种加成
最小伤害 = 1
```

### 伤害浮动
```
实际伤害 = 基础伤害 × (0.9 ~ 1.1随机浮动)
```

## 六、敌人设计

### 敌人类型

| 类型 | HP | 攻击 | 防御 | 速度 | 特点 |
|------|-----|------|------|------|------|
| 小怪 | 50 | 10 | 3 | 90 | 纯攻击 |
| 精英 | 80 | 15 | 5 | 100 | 每3回合蓄力 |
| Boss | 150 | 20 | 8 | 80 | 阶段变身、蓄力大招 |
| 召唤师 | 60 | 12 | 4 | 110 | 召唤小怪 |

### 蓄力机制

敌人蓄力时：
1. 行动条显示 ⚠️ 标记
2. 头像上方显示技能名和伤害预览
3. 蓄力期间玩家可以：
   - 用控制技能打断
   - 用防御技能减伤
   - 用爆发技能先秒掉

### 蓄力技能示例

| 技能名 | 蓄力回合 | 效果 |
|--------|----------|------|
| 致命打击 | 1回合 | 造成80伤害 |
| 全体攻击 | 2回合 | 对全体造成30伤害 |
| 召唤 | 1回合 | 召唤2只小怪 |
| 狂暴 | 2回合 | 攻击力+50% |

## 七、行动条系统

### 行动条规则
1. 按速度值从高到低排序
2. 当前行动角色头像放大+发光
3. 已行动角色头像变灰
4. 每回合重新排序（考虑速度变化）

### 行动条UI
```
┌─────────────────────────────────────┐
│  [星祭司]→[摸金手]→[敌人]→[搬山客]→[铁柱]→[敌人]
│    ↑当前                           │
└─────────────────────────────────────┘
```

## 八、插队机制

### 插队规则
1. 怒气满时英雄头像发光
2. 玩家点击可立即释放技能
3. 释放后怒气归零
4. 不影响行动条顺序
5. 可连续插队（多个英雄怒气满时）

### 插队示例
```
行动条：星祭司(当前) → 摸金手 → 敌人 → 搬山客

摸金手怒气满了：
→ 玩家点击摸金手 → 摸金手插队放技能
→ 行动条继续：星祭司(当前) → 摸金手 → 敌人 → 搬山客(已过)
```

## 十一、疲劳度系统

### 规则
- 关卡初始疲劳度：100
- 战斗消耗：损失的总HP = 疲劳度消耗
- HP每场战斗后自动回满
- 疲劳度归零时无法继续战斗

### 完美通关设计
- 5只小怪 + 1只Boss
- 预计消耗：约99点疲劳度
- 完美通关剩余：1-3点

### 通用动效规范

#### 伤害飘字
```
颜色：红色（普通）、黄色（暴击）、绿色（治疗）
字体：粗体，带描边
动画：从目标头顶生成 → 上浮50px → 淡出消失
时长：0.8秒
```

#### 受击反馈
```
普通受击：头像左右抖动3次，红色闪烁
暴击受击：更大幅度抖动，黄色闪光
死亡：头像变灰 → 缩小淡出 → 粒子消散
```

#### 怒气满
```
头像边缘：金色光晕脉冲（1秒周期）
图标：星星闪烁
提示文字："★ 可放技能"
```
