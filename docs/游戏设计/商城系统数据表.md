## 一、设计概述

### 1.1 商城系统架构

```
商城系统
├── 商品配置（商品定义）
├── 商品分类（分类管理）
├── 购买记录（交易历史）
└── 限购配置（购买限制）
```

### 1.2 表清单

| 序号 | 表名 | 说明 |
|------|------|------|
| 1 | shop_items | 商城商品配置表 |
| 2 | shop_categories | 商品分类表 |
| 3 | user_shop_logs | 购买记录表 |
| 4 | shop_limit_config | 限购配置表 |
| 5 | user_purchase_records | 玩家购买记录表 |

## 八、常用查询示例

### 8.1 查询商城商品列表

```sql
SELECT si.*, sc.category_name
FROM shop_items si
JOIN shop_categories sc ON si.category_id = sc.id
WHERE si.is_active = 1 
    AND si.show_in_store = 1
    AND (si.start_time IS NULL OR si.start_time <= NOW())
    AND (si.end_time IS NULL OR si.end_time >= NOW())
ORDER BY sc.sort_order, si.sort_order;
```

### 8.2 查询玩家购买状态

```sql
SELECT 
    si.id as shop_item_id,
    si.item_name,
    si.limit_type,
    si.limit_count,
    COALESCE(sl.purchase_count, 0) as purchased_count,
    si.limit_count - COALESCE(sl.purchase_count, 0) as remain_count,
    CASE WHEN COALESCE(sl.purchase_count, 0) >= si.limit_count THEN 0 ELSE 1 END as can_buy
FROM shop_items si
LEFT JOIN (
    SELECT shop_item_id, SUM(purchase_count) as purchase_count
    FROM user_purchase_records
    WHERE user_id = ? AND reset_flag = CONCAT('daily_', CURDATE())
    GROUP BY shop_item_id
) sl ON si.id = sl.shop_item_id
WHERE si.is_active = 1 AND si.limit_type = 'daily';
```

### 8.3 查询玩家累计消费

```sql
SELECT 
    SUM(cost_amount) as total_spent,
    COUNT(*) as purchase_count
FROM user_shop_logs
WHERE user_id = ? AND payment_status = 2;
```
