## 一、第一天英雄获取总览

### 1.0 玩家身份与开场设定

**核心设定**：
- 玩家**本身就是一个英雄**，可以加入战斗
- 玩家能力**偏向中庸型**，通过时装改变定位
- 玩家是**四大传承的共同后人**（设定层面，不与玩法绑定）
- 新手阶段**不需要选择传承**

**玩家真实身份**（逐步揭示）：
- 第1层：四大传承的共同后人
- 第2层：上古星祭师的转世
- 第3层：**紫微星核的守护者**
- 第4层：**苍穹劫的关键人物**

**游戏开场：苏醒于棺椁**

玩家从棺椁中苏醒，被陈星垣发现：

```
游戏开场流程：

【黑屏】
  画外音（陈星垣）： "这是...什么情况？"
      ↓
【画面渐亮】
  场景：古老墓室，昏暗火把
  陈星垣站在石棺前，石棺盖被推开
      ↓
【玩家视角】
  从棺椁中坐起，视角模糊
  陈星垣： "你...你是谁？你怎么会在棺椁里？"
      ↓
【选择回答】
  选项A： "我不知道..."
  选项B： "这里是哪里？"
  选项C： "你是谁？"
      ↓
【剧情推进】
  陈星垣发现玩家身上有星纹印记
  解释现状、自己的身份、古墓情况
      ↓
【获得初始伙伴】
  陈星垣： "这座古墓很危险，你需要伙伴。"
  第一个英雄登场
      ↓
【共同逃离】
  两人开始探索古墓，寻找出口
      ↓
【玩家参与战斗】
  玩家作为英雄参与第一场战斗
```

### 1.1 英雄获取总览

| 时机 | 英雄 | 品质 | 获取方式 | 目的 |
|------|------|------|----------|------|
| 游戏开始 | **玩家英雄** | 特殊 | 自动拥有 | 中庸型，可变定位 |
| 新手引导第1关 | 铁柱 | R | 固定赠送 | 坦克定位 |
| 第1关完成后 | 随机N/R | N/R | 新手10连 | 阵容补充 |

**第一天结束状态**：玩家拥有 3~4 个英雄（玩家英雄 + 2~3个其他英雄）

```

│     ↓                                                   │

| 定位 | 中庸型 / 全能型 |
| 基础属性 | HP 100 / ATK 22 / DEF 6 / SPD 100 |
| 技能 | 星光一击（普攻）/ 星陨（怒攻）/ 星纹护体（被动） |

```

│                                                         │
│  步骤1：玩家苏醒                                         │

│  步骤2：进入第1关「古老墓穴」                            │
│     ↓                                                   │

**触发条件**：完成第1关战斗教学
**获得英雄**：铁柱（R品质）
**获得方式**：系统赠送

**铁柱详细信息**：

| 项目 | 内容 |
|------|------|
| 名称 | 铁柱 |
| 品质 | R（星品） |
| 职业 | 重装战士 |
| 传承 | 天枢发丘 |
| 定位 | 坦克/前排 |
| 基础属性 | HP 120 / ATK 20 / DEF 8 / SPD 85 |
| 技能 | 冲锋（怒攻）：1.5倍伤害+眩晕 |

**设计理由**：
- 坦克定位，与玩家英雄（中庸型）互补
- 高血量高防御，适合新手

**触发条件**：完成第1关，开放抽卡系统

| 保底 | 说明 |
|------|------|
| 必出N | 至少1个N品质英雄 |
| 必出R | 至少1个R品质英雄 |
| 概率SR | 8%概率获得SR |

## 三、第一天阵容规划

### 3.1 玩家阵容组成

```
┌─────────────────────────────────────────┐
│  Day 1 玩家阵容                         │
├─────────────────────────────────────────┤
│                                         │
│  前排：铁柱（R）- 坦克                   │
│  中排：玩家英雄 - 中庸型                 │
│  后排：随机R英雄 - 输出/辅助             │
│                                         │
│  总计：3~4个英雄                         │
│  组成：玩家英雄 + 铁柱 + 随机英雄        │
│                                         │
└─────────────────────────────────────────┘
```

**阵容搭配建议**：
- 玩家英雄必带（中庸型，适应性强）
- 铁柱必带（坦克，保护后排）
- 随机英雄根据抽到的类型搭配

### 3.2 玩家英雄在阵容中的角色

| 搭配方案 | 玩家英雄定位 | 说明 |
|----------|--------------|------|
| 缺输出 | 主力输出 | 玩家英雄攻击中等，可补充输出 |
| 缺坦克 | 副坦克 | 玩家英雄防御中等，可分担伤害 |
| 缺辅助 | 辅助位 | 玩家英雄均衡，可提供支援 |
