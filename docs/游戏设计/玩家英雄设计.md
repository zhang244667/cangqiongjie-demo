## 一、玩家英雄概述

**核心定位**：玩家本身就是一个可战斗的英雄

| 项目 | 内容 |
|------|------|
| 名称 | 寻龙者（玩家自定义） |
| 品质 | 特殊（独立成长系统） |
| 定位 | 中庸型 / 全能型 |
| 特点 | 能力均衡，可通过时装改变定位 |

**设计原则**：
- 玩家英雄**不参与抽卡系统**，是固定拥有的
- 玩家英雄**成长独立**，不占用其他英雄资源
- 玩家英雄**时装系统**改变能力和定位
- 玩家英雄**中庸型**设计，不突出但也不弱

## 二、玩家英雄属性

### 2.1 基础属性（中庸型）

| 属性 | 数值 | 对比其他英雄 |
|------|------|--------------|
| HP | 100 | 中等（坦克120，法师75） |
| 攻击 | 22 | 中等（输出24，坦克20） |
| 防御 | 6 | 中等（坦克8，法师4） |
| 速度 | 100 | 中等（敏捷110，坦克85） |

**属性特点**：
- 无明显短板，也无突出优势
- 适应性强，可配合各种阵容
- 通过时装可强化某个方向

### 2.2 属性成长

| 等级 | HP | 攻击 | 防御 | 速度 |
|------|-----|------|------|------|
| 1级 | 100 | 22 | 6 | 100 |
| 50级 | 250 | 55 | 15 | 100 |
| 100级 | 500 | 110 | 30 | 100 |
| 200级 | 1000 | 220 | 60 | 100 |
| 300级 | 1500 | 330 | 90 | 100 |

**成长系数**：
- HP成长：每级 +5
- 攻击成长：每级 +1.1
- 防御成长：每级 +0.3
- 速度：固定不变

## 三、玩家英雄技能

### 3.1 技能设计原则

| 原则 | 说明 |
|------|------|
| 基础技能 | 固定3个技能（普攻/怒攻/被动） |
| 时装技能 | 穿戴时装后技能变化 |
| 逐步解锁 | 随等级解锁技能 |

### 3.2 基础技能（默认）

**技能1：普通攻击 - 星光一击**

```
技能名称：星光一击
技能类型：普攻
效果描述：对单体敌人造成100%攻击伤害
冷却时间：无
解锁条件：默认拥有
```

**技能2：怒气技能 - 星陨**

```
技能名称：星陨
技能类型：怒攻
效果描述：召唤星光坠落，对全体敌人造成120%攻击伤害
怒气消耗：100
冷却时间：无
解锁条件：玩家等级10级
```

**技能3：被动技能 - 星纹护体**

```
技能名称：星纹护体
技能类型：被动
效果描述：受到伤害时，有15%概率减免20%伤害
触发条件：被攻击时
解锁条件：玩家等级30级
```

### 3.3 技能成长

| 等级 | 普攻伤害 | 怒攻伤害 | 被动触发率 |
|------|----------|----------|------------|
| 1级 | 100% | 120% | 15% |
| 50级 | 115% | 140% | 20% |
| 100级 | 130% | 160% | 25% |
| 200级 | 150% | 180% | 30% |
| 300级 | 170% | 200% | 35% |

## 四、时装系统

### 4.1 时装概述

**时装定义**：
- 玩家英雄专属的外观+能力套装
- 穿戴时装后，玩家英雄的**属性、技能、定位**都会改变
- 时装通过活动、商城、成就等方式获取

**时装功能**：
| 功能 | 说明 |
|------|------|
| 外观改变 | 更换玩家英雄的外观形象 |
| 属性加成 | 改变玩家英雄的属性分布 |
| 技能替换 | 替换玩家英雄的技能 |
| 定位转变 | 改变玩家英雄的战斗定位 |

### 4.2 时装分类

**按定位分类**：

| 时装类型 | 定位 | 属性特点 | 技能特点 |
|----------|------|----------|----------|
| 战士时装 | 前排输出 | 高攻击、高HP | 单体爆发、吸血 |
| 坦克时装 | 前排坦克 | 高HP、高防御 | 嘲讽、护盾 |
| 法师时装 | 后排法师 | 高攻击、低HP | 群体伤害、控制 |
| 辅助时装 | 后排辅助 | 均衡属性 | 治疗、增益 |

## 五、玩家英雄获取与成长

### 5.1 获取方式

| 获取时机 | 内容 |
|----------|------|
| 游戏开始 | 玩家英雄自动拥有，无需获取 |
| 初始状态 | 1级，默认技能，无时装 |
| 首充 | 获得时装「破阵先锋」 |

### 5.2 成长系统

| 系统 | 说明 |
|------|------|
| 等级提升 | 通过战斗和道具升级 |
| 技能升级 | 随等级自动提升 |
| 时装收集 | 通过活动/商城获取时装 |
| 时装强化 | 提升时装的属性加成 |

### 5.3 玩家英雄与普通英雄的区别

| 项目 | 玩家英雄 | 普通英雄 |
|------|----------|----------|
| 获取方式 | 自动拥有 | 抽卡/碎片合成 |
| 品质系统 | 无品质，独立成长 | N/R/SR/SSR/UR |
| 技能变化 | 通过时装改变 | 固定技能 |
| 定位变化 | 时装改变定位 | 固定定位 |
| 成长资源 | 独立资源池 | 共享资源池 |

## 七、设计总结

**玩家英雄定位**：

| 特点 | 说明 |
|------|------|
| 中庸型 | 属性均衡，无短板无优势 |
| 灵活多变 | 通过时装改变定位 |
| 独立成长 | 不占用其他英雄资源 |
| 专属系统 | 时装系统独占 |

**设计意义**：

| 意义 | 说明 |
|------|------|
| 玩家代入感 | 玩家本身参与战斗，代入感更强 |
| 商业化 | 时装系统是重要付费点 |
| 玩法多样 | 一套属性多种玩法 |
| 收集驱动 | 收集时装成为长期目标 |
