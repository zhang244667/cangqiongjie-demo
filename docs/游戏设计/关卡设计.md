## 二、第1关：古老墓穴

### 2.1 关卡概述

| 项目 | 内容 |
|------|------|
| 地图类型 | 随机生成 |
| 地图大小 | 20×20 |
| 难度 | 入门 |
| 推荐等级 | 1-10级 |

### 2.2 地图生成规则

- **墙壁**：地图边缘为墙壁，内部随机生成墙壁（密度约15%）
- **怪物**：随机分布，数量 3-5 只「墓穴守卫」
- **宝箱**：随机分布，数量 2-3 个
- **机关**：三种类型各 1-2 个
- **出口**：随机放置，距起点至少 10 格

### 2.3 事件分布

```
| 事件类型 | 数量 | 概率 |
|----------|------|------|
| 怪物(enemy) | 3-5 | 5% |
| 宝箱(chest) | 2-3 | 4% |
| 运气机关 | 1-2 | 4% |
| 知识机关 | 1-2 | 4% |
| 英雄机关 | 1-2 | 4% |
```

### 4.2 地图布局（16×16 格子）

#### 坐标说明
- 坐标系：`(x, y)`，左上角为 `(0, 0)`
- `W` = 墙壁
- `·` = 地板（可通行）
- `P` = 玩家起点
- `E` = 敌人
- `B` = Boss
- `S` = 出口/楼梯
- `A` = 攻击Buff
- `R` = 休息点
- `1` = 机关类型1（运气型）
- `2` = 机关类型2（知识型）
- `3` = 机关类型3（英雄型）

### 4.5 流程设计

```
[入口]
    ↓
[走廊A：拦路怪×2] → 触发战斗
    ↓
[前厅：运气机关] → 触发机关互动
    ↓
[通道B：拦路怪×2] → 触发战斗
    ↓
[走廊C：可选探索]
    ↓
[Boss房前] ← 可以先去拿攻击Buff
    ↓
[Boss房：星棺守卫] → 触发Boss战
    ↓
[击败Boss后解锁出口]
    ↓
[休息点（可选）] → 恢复疲劳
    ↓
[出口] → 通关结算
```
