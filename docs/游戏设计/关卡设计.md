## 二、第1关：古老墓穴

### 2.1 关卡概述

| 项目 | 内容 |
|------|------|
| 地图类型 | 随机生成 |
| 地图大小 | 20×20 |
| 难度 | 入门 |
| 推荐等级 | 1-10级 |

### 2.2 地图生成规则

- **墙壁**：地图边缘为墙壁，内部随机生成墙壁（密度约15%）
- **怪物**：随机分布，数量 3-5 只「墓穴守卫」
- **宝箱**：随机分布，数量 2-3 个
- **机关**：三种类型各 1-2 个
- **出口**：随机放置，距起点至少 10 格

### 2.3 事件分布

```
| 事件类型 | 数量 | 概率 |
|----------|------|------|
| 怪物(enemy) | 3-5 | 5% |
| 宝箱(chest) | 2-3 | 4% |
| 运气机关 | 1-2 | 4% |
| 知识机关 | 1-2 | 4% |
| 英雄机关 | 1-2 | 4% |
```

## 四、第3关：星陨古墓（线性Boss关）

### 4.1 关卡概述

| 项目 | 内容 |
|------|------|
| 地图类型 | **固定线性地图** |
| 地图大小 | 20×16 |
| 难度 | Boss战 |
| 推荐等级 | 20-30级 |
| 特殊 | 必须击败星棺守卫才能通关 |

### 4.2 设计原则

- **线性空间结构**：每个区域用墙壁完全围起，只留必要出入口
- **强制流程**：玩家必须按顺序通过所有区域
- **无自由探索**：无回头路，不可跳关

### 4.3 区域规划

| 区域 | 坐标范围 | 大小 | 内容 | 说明 |
|------|----------|------|------|------|
| 入口区 | (1,1)-(2,2) | 2×2 | 玩家起点 P | 初始位置 |
| 走廊A | (3,1)-(7,3) | 5×3 | 敌人 E×2 | 第一道拦路怪，必须战斗通过 |
| 前厅 | (8,1)-(12,4) | 5×4 | 运气机关 1 | 宝箱机关 |
| 通道B | (8,5)-(12,9) | 5×5 | 敌人 E×2 | 第二道拦路怪 |
| Boss前厅 | (8,10)-(11,12) | 4×3 | - | Boss战准备区 |
| Boss房 | (6,13)-(11,15) | 6×3 | 星棺守卫 B | 最终Boss战 |
| 休息点 | (12,14)-(13,15) | 2×2 | 疲劳恢复 R | 恢复30点疲劳 |
| 出口 | (16,15) | 1×1 | 楼梯 S | 通关目标 |

### 4.5 流程设计（线性推进）

```
[入口区]
    ↓
[走廊A：拦路怪×2] → 必须击败敌人才能继续
    ↓
[前厅：运气机关] → 触发星辰宝箱机关
    ↓
[通道B：拦路怪×2] → 必须击败敌人才能继续
    ↓
[Boss前厅] → 可选择攻击Buff
    ↓
[Boss房：星棺守卫] → 触发Boss战，必须击败Boss
    ↓
[休息点] → 可选，恢复疲劳
    ↓
[出口] → 通关结算
```
