## 二、普攻伤害公式

### 2.1 普攻基础伤害

```
NormalAttackDamage = ATK × NormalAttackCoeff × (1 - EnemyDEF_Reduction)
```

### 2.2 普攻技能系数

| 品质 | 普攻系数 | 说明 |
|------|----------|------|
| 凡品(N) | 100% | 基础 |
| 灵品(R) | 105% | +5% |
| 星品(SR) | 110% | +10% |
| 月品(SSR) | 115% | +15% |
| 日品(UR) | 120% | +20% |

### 2.3 普攻技能等级加成

每级提升伤害：

```
NormalAttackCoeff(level) = BaseCoeff + (level - 1) × 0.05
```

| 技能等级 | 系数加成 | 总系数(SSR) |
|----------|----------|-------------|
| 1 | +0% | 115% |
| 5 | +20% | 135% |
| 10 | +45% | 160% |
| 15 | +70% | 185% |
| 20 | +95% | 210% |

### 2.4 普攻伤害示例

**双钩定（SSR），ATK=5000，普攻10级**
- 基础系数：115%
- 等级加成：45%
- 最终系数：160%

```
NormalDamage = 5000 × 1.60 × (1 - 0.5) = 4000
```

## 三、怒攻伤害公式

### 3.1 怒攻基础伤害

```
SkillDamage = ATK × SkillAttackCoeff × SkillLevelBonus × (1 - EnemyDEF_Reduction)
```

### 3.2 怒攻技能系数

| 品质 | 怒攻基础系数 | 说明 |
|------|--------------|------|
| 凡品(N) | 150% | 基础 |
| 灵品(R) | 160% | +10% |
| 星品(SR) | 175% | +25% |
| 月品(SSR) | 200% | +50% |
| 日品(UR) | 250% | +100% |

### 3.3 怒攻技能等级加成

```
SkillAttackCoeff(level) = BaseCoeff + (level - 1) × 0.10
```

| 技能等级 | 系数加成 | 总系数(SSR) |
|----------|----------|-------------|
| 1 | +0% | 200% |
| 5 | +40% | 240% |
| 10 | +90% | 290% |
| 15 | +140% | 340% |
| 20 | +190% | 390% |

### 3.4 怒攻伤害示例

**双钩定（SSR），ATK=5000，怒攻15级**
- 基础系数：200%
- 等级加成：140%
- 最终系数：340%

```
SkillDamage = 5000 × 3.40 × (1 - 0.5) = 8500
```

## 六、伤害附加效果公式

### 6.1 附加伤害公式

```
ExtraDamage = BaseDamage × ExtraDamageRate × SkillLevelBonus
```

### 6.2 附加效果触发率

```
TriggerChance = BaseChance + LevelBonus × (1 + BonusRate)
```

| 附加效果 | 基础触发率 | 每级提升 |
|----------|------------|----------|
| 灼烧(每回合伤害) | 30% | +3% |
| 中毒(持续伤害) | 25% | +3% |
| 眩晕(回合跳过) | 15% | +2% |
| 破甲(防御降低) | 20% | +2% |

### 6.3 持续伤害公式

```
DotDamage = ATK × DotCoeff × (1 + SkillLevelBonus) × Turns
```

| 持续效果 | 每回合伤害系数 | 持续回合 |
|----------|---------------|----------|
| 灼烧 | 5% ATK | 2-3回合 |
| 中毒 | 8% ATK | 3回合 |
| 诅咒 | 3% HP | 2回合 |

## 七、治疗效果公式

### 7.1 基础治疗公式

```
Healing = BaseHealValue × (1 + HealBonus) × SkillLevelBonus
```

### 7.2 治疗效果配置

| 技能类型 | 基础治疗量 | 每级提升 |
|----------|------------|----------|
| 百分比治疗 | 目标HP × 10% | +2% |
| 固定治疗 | 治疗者ATK × 50% | +10% |
| 生命吸取 | 伤害量 × 30% | +3% |

### 7.3 治疗示例

**技能：摸金回春**
- 效果：治疗全队，治疗量为施法者攻击的100%
- 每级提升：+15%

```
治疗计算（ATK=3000，技能5级）：
- 基础治疗 = 3000 × 1.0 = 3000
- 等级加成 = 1 + 4 × 0.15 = 1.6
- 最终治疗 = 3000 × 1.6 = 4800
```

## 八、护盾效果公式

### 8.1 护盾吸收量

```
ShieldValue = BaseShield × (1 + ShieldBonus) × SkillLevelBonus
```

### 8.2 护盾配置

| 护盾类型 | 基础值 | 每级提升 | 持续回合 |
|----------|--------|----------|----------|
| 普通护盾 | ATK × 100% | +20% | 2回合 |
| 高级护盾 | ATK × 150% | +30% | 3回合 |
| 绝对护盾 | HP × 20% | +5% | 1回合 |

### 8.3 护盾优先级

```
ShieldDamage = Min(ShieldValue, DamageReceived)
RemainingDamage = Max(0, DamageReceived - ShieldDamage)
```

## 九、怒气技能效果公式

### 9.1 怒气消耗规则

| 技能类型 | 怒气消耗 | 可被打断 |
|----------|----------|----------|
| 普通怒攻 | 100 | 否 |
| 终极怒攻 | 150 | 否 |
| 被动触发 | 0 | - |

### 9.2 怒气减免效果

```
ActualAngerCost = BaseCost × (1 - AngerCostReduction)
```

### 9.3 怒气回复效果

```
AngerGain = BaseGain × (1 + AngerGainBonus) + KillBonus
```

## 十、特殊技能公式

### 10.1 复活技能

```
ReviveCondition = SkillLevel >= RequiredLevel AND ReviveCount < MaxReviveCount
RevivedHP = MaxHP × ReviveHPRatio
```

### 10.2 反击技能

```
CounterDamage = ReceivedDamage × CounterRatio × (1 + SkillLevelBonus)
```

### 10.3 溅射伤害

```
SplashDamage = MainTargetDamage × SplashRatio × SplashReduction
```

| 溅射目标 | 溅射比例 |
|----------|----------|
| 相邻单位 | 50% |
| 全体目标 | 30% |

## 十二、技能效果示例

### 12.1 完整怒攻计算

**技能：探墓绝杀（SSR）**
- 基础系数：200%
- 每级提升：10%
- 当前等级：15级
- 附加效果：30%概率灼烧

**攻击方：双钩定**
- ATK：5000
- 暴击率：40%
- 暴击伤害：180%

**防御方：古墓守卫**
- DEF：1500（减免=60%）

**计算过程：**

```
1. 技能系数 = 200% + 14 × 10% = 340%

2. BaseDamage = 5000 × 3.40 × (1 - 0.6) = 6800

3. 暴击判定（40%）：假设触发暴击
   CriticalDamage = 6800 × 1.80 = 12240

4. 灼烧判定（30%）：假设触发
   DotDamage = 5000 × 5% × 2 = 500/回合

5. 最终伤害 = 12240 + 500 = 12740
```
