# 《苍穹劫》英雄养成系统设计

> 核心设计文档 - 版本 2.0
> 更新日期：2026年4月23日

---

## 一、养成系统总览

### 1.1 养成分类

| 类型 | 系统 | 是否可转移 | 说明 |
|------|------|------------|------|
| **英雄绑定** | 等级 | ❌ 不可 | 消耗经验提升，上限300级 |
| **英雄绑定** | 突破 | ❌ 不可 | 6阶突破，解锁等级上限 |
| **英雄绑定** | 星级 | ❌ 不可 | 消耗碎片升星，6星满星 |
| **英雄绑定** | 技能 | ❌ 不可 | 随突破自动提升 |
| **可转移** | 宝物 | ✅ 可 | 每英雄1槽，盗墓宝物 |

### 1.2 设计原则

- **多英雄横向养成友好**：每个英雄养成点精简，避免单英雄养成过重
- **宝物可转移**：降低养成成本，方便阵容调整
- **技能自动提升**：减少养成维度，聚焦核心养成点

## 二、英雄等级与突破

### 2.1 等级系统

| 维度 | 设计 |
|------|------|
| 等级上限 | 300级 |
| 升级消耗 | 经验药水（小/中/大） |
| 升级效果 | 基础属性提升（攻击、生命、防御、速度） |

**经验来源**：
- 地宫探索（主要）
- 经验副本
- 日常任务
- 活动奖励

### 2.2 突破系统

**核心机制**：等级达到突破节点时，必须突破才能继续升级

| 突破阶段 | 突破等级 | 升级跨度 | 突破消耗 | 突破效果 |
|----------|----------|----------|----------|----------|
| 1阶 | 30级 | 30级 | 突破石x30 + 金币 | 解锁30-70级，普攻等级+1 |
| 2阶 | 70级 | 40级 | 突破石x60 + 金币 | 解锁70-120级，被动效果+20% |
| 3阶 | 120级 | 50级 | 突破石x120 + 传承材料 | 解锁120-180级，怒攻效果+30% |
| 4阶 | 180级 | 60级 | 突破石x200 + 传承材料 | 解锁180-250级，全技能+50% |
| 5阶 | 250级 | 70级 | 突破石x350 + 稀有材料 | 解锁250-300级，解锁技能特效 |
| 6阶 | 300级 | - | - | 满级成就，专属称号 |

**突破节奏**：
- 前期（30级）：1-3天可达，快速体验突破
- 中前期（70级）：1-2周，建立养成习惯
- 中期（120级）：1-2月，核心养成期
- 中后期（180级）：3-4月，深度养成
- 后期（250级）：6月+，长期目标
- 终极（300级）：1年+，满级追求

## 三、技能系统

### 3.1 技能结构

**统一3技能**：所有英雄固定3个技能

| 技能类型 | 说明 | 使用方式 |
|----------|------|----------|
| **普攻** | 基础攻击技能 | 无消耗，每回合可用 |
| **怒攻** | 消耗怒气的强力技能 | 消耗怒气，高伤害/特殊效果 |
| **被动** | 自动生效的效果 | 无需操作，持续生效 |

### 3.2 品质差异

品质差异体现在**技能强度和被动效果**，而非技能数量

| 品质 | 普攻 | 怒攻 | 被动 |
|------|------|------|------|
| N（凡品） | 基础伤害 | 基础效果 | 单一效果 |
| R（灵品） | +10%伤害 | +15%效果 | 效果增强 |
| SR（星品） | +20%伤害 | +30%效果 | 双重效果 |
| SSR（月品） | +35%伤害 | +50%效果 | 强力效果 |
| UR（日品） | +50%伤害 | +80%效果 | 独特机制 |

### 3.3 技能随突破提升

技能不再单独养成，随突破自动提升：

| 突破阶段 | 技能提升 |
|----------|----------|
| 1阶 | 普攻等级+1 |
| 2阶 | 被动效果+20% |
| 3阶 | 怒攻等级+2 |
| 4阶 | 全技能等级+3 |
| 5阶 | 解锁技能特效变化 |

## 四、星级系统

### 4.1 升星机制

消耗同名英雄碎片升星，属性大幅提升

| 星级 | 升星消耗 | 属性加成 | 解锁内容 |
|------|----------|----------|----------|
| ★ 1星 | 初始 | 基础 | 基础技能 |
| ★★ 2星 | 20碎片 | +20% | 被动效果+10% |
| ★★★ 3星 | 40碎片 | +40% | 被动效果强化 |
| ★★★★ 4星 | 80碎片 | +60% | 怒攻效果+20% |
| ★★★★★ 5星 | 150碎片 | +80% | 解锁觉醒外观 |
| ★★★★★★ 6星 | 300碎片 | +120% | 终极形态，专属特效 |

### 4.2 碎片获取

| 获取方式 | 说明 |
|----------|------|
| 抽卡重复 | 抽到重复英雄转化为碎片 |
| 关卡掉落 | 特定关卡掉落英雄碎片 |
| 商店兑换 | 各类商店兑换指定碎片 |
| 活动奖励 | 限时活动获取 |

## 五、宝物系统

### 5.1 基本设计

| 维度 | 设计 |
|------|------|
| 槽位数量 | 每英雄1个 |
| 可否转移 | ✅ 可拆卸、可转移 |
| 宝物等级上限 | 50级 |
| 突破次数 | 5次（每10级突破） |

### 5.2 宝物类型

| 类型 | 宝物示例 | 主属性方向 |
|------|----------|------------|
| 法器类 | 发丘印、摸金符、罗盘、符笔 | 攻击、技能效果 |
| 防护类 | 黑驴蹄子、糯米、朱砂 | 生存、抗性 |
| 兵器类 | 工兵铲、登山镐、金刚伞 | 攻击、破甲 |
| 古董类 | 青铜镜、玉璧、金缕玉衣 | 均衡、特殊效果 |
| 秘宝类 | 尸香魔芋、鬼眼、长生烛 | 特殊机制 |

### 5.3 宝物品质

| 品质 | 名称 | 属性系数 | 获取方式 |
|------|------|----------|----------|
| N | 凡品 | 0.7x | 普通关卡 |
| R | 灵品 | 0.85x | Boss关卡 |
| SR | 星品 | 1.0x | 深渊/活动 |
| SSR | 月品 | 1.2x | 限时活动 |
| UR | 日品 | 1.5x | 年度活动 |

### 5.4 宝物养成

#### 强化

| 维度 | 设计 |
|------|------|
| 消耗 | 其他宝物 或 宝物经验石 |
| 效果 | 提升宝物主属性 |
| 上限 | 当前突破阶段上限 |

#### 突破

| 突破阶段 | 突破等级 | 突破消耗 | 效果 |
|----------|----------|----------|------|
| 1阶 | 10级 | 同类宝物x1 + 金币 | 解锁10-20级，主属性+10% |
| 2阶 | 20级 | 同类宝物x2 + 金币 | 解锁20-30级，解锁副属性 |
| 3阶 | 30级 | 同类宝物x3 + 材料 | 解锁30-40级，主属性+20% |
| 4阶 | 40级 | 同类宝物x5 + 材料 | 解锁40-50级，副属性+30% |
| 5阶 | 50级 | 同类宝物x8 + 稀有材料 | 满级，解锁宝物特效 |

## 六、养成节奏

### 6.1 前期（1-30天）

| 养成重点 | 目标 |
|----------|------|
| 等级 | 主力英雄拉到70-120级 |
| 突破 | 完成1-2阶突破 |
| 星级 | 主力2-3星 |
| 宝物 | 给主力装备灵品宝物 |

### 6.2 中期（30-180天）

| 养成重点 | 目标 |
|----------|------|
| 等级 | 主力180-250级，培养第二梯队 |
| 突破 | 主力4-5阶突破 |
| 星级 | 主力4-5星 |
| 宝物 | 星品宝物+强化+突破 |

### 6.3 后期（180天+）

| 养成重点 | 目标 |
|----------|------|
| 等级 | 全英雄300级 |
| 突破 | 主力6阶满突破 |
| 星级 | 主力6星 |
| 宝物 | 月品/日品宝物+满突破 |

## 七、关键决策记录

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 技能数量 | 统一3个 | 简化养成，降低理解成本 |
| 技能养成 | 随突破提升 | 减少养成维度 |
| 等级上限 | 300级 | 数字感更强，长期追求 |
| 突破节奏 | 6阶（前快后慢） | 前期爽感，后期长期目标 |
| 宝物槽位 | 每英雄1个 | 方便多英雄横向养成 |
| 宝物转移 | 可转移 | 降低养成成本，阵容调整友好 |
