# 竞技场系统设计文档（离线匹配模式）

## 一、系统概述

### 1.1 系统定位
竞技场系统是《苍穹劫·摸金传人》的核心PVP玩法，采用**离线匹配模式**，玩家从系统预设的对手列表中直接选择挑战。系统通过段位机制、赛季制度和丰富奖励，驱动玩家持续参与PVP竞争。

### 1.2 设计目标
- **简洁高效**：无需等待，即刻挑战
- **公平竞技**：确保对战双方实力接近
- **策略对抗**：深度技能组合与战斗策略
- **目标驱动**：段位提升提供长期追求
- **激励循环**：胜利奖励与赛季奖励

### 1.3 核心变化（简化说明）
| 原设计 | 简化后 |
|--------|--------|
| 实时在线匹配 | 离线对手列表 |
| 匹配等待时间 | 直接选择对手 |
| 好友切磋模式 | 已移除 |
| 观战系统 | 已移除 |
| 复杂匹配算法 | 智能对手生成 |

## 六、积分计算规则

### 6.1 积分公式

```yaml
积分计算公式:

  基础积分:
    胜利: +20分
    失败: -10分
    
  段位加成:
    青铜: ×1.0
    白银: ×1.1
    黄金: ×1.2
    钻石: ×1.3
    大师: ×1.4
    宗师: ×1.5
    王者: ×1.6
    
  连胜加成:
    3连胜: +5额外积分
    5连胜: +10额外积分
    10连胜: +20额外积分
    
  积分上限:
    每日最大获得: 200分
    每日最大损失: 100分
```

### 6.2 积分计算示例

```
示例1：钻石段位玩家胜利
  基础积分: 20分
  段位加成: ×1.3
  结果: 20 × 1.3 = 26分

示例2：大师段位玩家3连胜
  基础积分: 20分
  段位加成: ×1.4
  连胜加成: +5分
  结果: (20 × 1.4) + 5 = 33分

示例3：宗师段位玩家失败
  基础积分: -10分
  段位加成: ×1.5
  结果: -10 × 1.5 = -15分
```

## 十、已移除功能说明

以下为原设计中已移除的功能：

| 移除功能 | 原设计描述 | 移除原因 |
|---------|-----------|---------|
| 实时匹配 | 等待在线玩家匹配 | 离线模式无需等待 |
| 匹配等待界面 | 显示匹配进度和倒计时 | 直接显示对手列表 |
| 好友切磋 | 邀请好友进行友谊赛 | 简化设计 |
| 观战系统 | 观看其他玩家对战 | 简化设计 |
| 匹配池 | Redis存储等待匹配玩家 | 不适用于离线模式 |
| 异常匹配检测 | 同IP多账号检测 | 离线模式无实时匹配 |
