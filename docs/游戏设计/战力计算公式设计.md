## 一、设计概述

### 1.1 战力体系说明

战力（PowerScore）是衡量英雄/宝物强度的综合指标，用于：
- 玩家阵容强度展示
- 匹配对手实力
- 排行榜排序
- 任务完成条件

### 1.2 战力构成

```
总战力 = 基础属性战力 + 技能战力 + 宝物战力 + 星级战力 + 突破战力
```

---

## 二、基础属性战力公式

### 2.1 属性战力权重表

各属性对战力的贡献权重：

| 属性 | 权重系数 | 说明 |
|------|----------|------|
| HP | 0.5 | 生命值权重 |
| Attack | 1.0 | 攻击力权重 |
| Defense | 0.8 | 防御力权重 |
| Speed | 0.6 | 速度权重 |

### 2.2 基础属性战力公式

```
BasePower = HP × W_HP + ATK × W_ATK + DEF × W_DEF + SPD × W_SPD
```

**简化公式：**

```
BasePower = HP × 0.5 + ATK × 1.0 + DEF × 0.8 + SPD × 0.6
```

### 2.3 基础属性战力示例

**双钩定（SR），150级，4星，3阶突破**
- HP = 64,034
- ATK = 3,500
- DEF = 1,200
- SPD = 480

```
BasePower = 64034 × 0.5 + 3500 × 1.0 + 1200 × 0.8 + 480 × 0.6
          = 32,017 + 3,500 + 960 + 288
          = 36,765
```

## 三、技能战力公式

### 3.1 技能战力权重

| 技能类型 | 基础战力 | 每级加成 |
|----------|----------|----------|
| 普攻 | 100 | +10 |
| 怒攻 | 200 | +25 |
| 被动 | 150 | +15 |

### 3.2 技能战力公式

```
SkillPower = Skill1_Power + Skill2_Power + Skill3_Power
```

**计算方式：**

```
SingleSkillPower = BaseSkillPower + SkillLevel × PerLevelBonus
```

### 3.3 技能战力示例

**双钩定，技能等级：普攻5/怒攻3/被动4**

```
普攻战力 = 100 + 5 × 10 = 150
怒攻战力 = 200 + 3 × 25 = 275
被动战力 = 150 + 4 × 15 = 210

总技能战力 = 150 + 275 + 210 = 635
```

## 四、宝物战力公式

### 4.1 宝物战力构成

```
ArtifactPower = MainStatPower + SubStatPower + SkillPower + QualityBonus
```

### 4.2 主属性战力

宝物主属性转换为战力：

```
MainStatPower = MainStatValue × StatPowerCoeff
```

- StatPowerCoeff = 50（属性战力系数）

### 4.3 主属性战力示例

**发丘印（SSR），50级，攻击+47.4%**

```
MainStatPower = 47.4 × 50 = 2,370
```

### 4.4 副属性战力

副属性解锁后计入战力：

```
SubStatPower = SubStatValue × StatPowerCoeff × 0.8
```

- 副属性系数 = 0.8（相对于主属性）

### 4.5 品质战力加成

| 品质 | 品质战力加成 |
|------|--------------|
| 凡品(N) | 0 |
| 灵品(R) | 500 |
| 星品(SR) | 1,500 |
| 月品(SSR) | 3,000 |
| 日品(UR) | 5,000 |

### 4.6 宝物战力示例

**发丘印（SSR），50级，5阶突破，副属性已解锁**

```
主属性战力 = 47.4 × 50 = 2,370
副属性战力 = 30 × 50 × 0.8 = 1,200
技能战力 = 500
品质加成 = 3,000

总宝物战力 = 2,370 + 1,200 + 500 + 3,000 = 7,070
```

## 五、星级战力公式

### 5.1 星级战力加成

每升一星获得固定战力加成：

| 星级 | HP战力加成 | ATK战力加成 | 额外战力 |
|------|------------|-------------|----------|
| 1→2 | +500 | +100 | +200 |
| 2→3 | +700 | +150 | +300 |
| 3→4 | +1000 | +200 | +400 |
| 4→5 | +1500 | +300 | +600 |
| 5→6 | +2000 | +400 | +800 |

### 5.2 星级累计战力

```
StarPower(star) = Σ(每级星级加成)
```

### 5.3 星级战力示例

**4星累计战力：**

```
StarPower(4星) = 0 + 800 + 1500 + 2400 = 4,700
```

## 七、综合战力计算

### 7.1 完整战力公式

```
TotalPower = BasePower 
           + SkillPower 
           + StarPower 
           + BreakthroughPower 
           + ArtifactBonusPower
```

### 7.2 战力计算流程

```
1. 计算基础属性战力
   BasePower = HP×0.5 + ATK×1.0 + DEF×0.8 + SPD×0.6

2. 计算技能战力
   SkillPower = Σ(单技能战力)

3. 计算星级战力
   StarPower = 累计星级战力加成

4. 计算突破战力
   BreakthroughPower = 累计突破战力加成

5. 计算宝物战力加成（装备该英雄时）
   ArtifactPower = 该宝物战力 × 0.5

6. 汇总
   TotalPower = Base + Skill + Star + Breakthrough + ArtifactPower
```

### 7.3 完整战力示例

**双钩定（SR）详细数据：**
- 等级：150
- HP：64,034，ATK：3,500，DEF：1,200，SPD：480
- 星级：4星
- 突破：3阶
- 技能：普攻5级/怒攻3级/被动4级
- 装备：发丘印（战力7,070）

**计算过程：**

```
1. 基础属性战力
   BasePower = 64034×0.5 + 3500×1.0 + 1200×0.8 + 480×0.6
             = 32,017 + 3,500 + 960 + 288
             = 36,765

2. 技能战力
   SkillPower = 150 + 275 + 210 = 635

3. 星级战力
   StarPower = 800 + 1500 + 2400 = 4,700

4. 突破战力
   BreakthroughPower = 1000 + 2000 + 4000 = 7,000

5. 宝物战力（装备加成50%）
   ArtifactPower = 7070 × 0.5 = 3,535

6. 总战力
   TotalPower = 36,765 + 635 + 4,700 + 7,000 + 3,535 = 52,635
```

## 九、战力排行榜

### 9.1 阵容战力计算

```
TeamPower = Σ(上阵英雄战力) + Σ(助战英雄战力 × 0.3)
```

### 9.2 战力分段说明

| 战力区间 | 强度评价 | 可通关章节 |
|----------|----------|------------|
| 0-10,000 | 新手 | 1-10章 |
| 10,000-50,000 | 进阶 | 11-30章 |
| 50,000-150,000 | 高手 | 31-60章 |
| 150,000-300,000 | 大神 | 61-90章 |
| 300,000+ | 巅峰 | 91-120章 |

## 十、战力使用指南

### 10.1 战力计算时机

| 场景 | 是否重新计算 |
|------|--------------|
| 英雄升级 | 是 |
| 英雄升星 | 是 |
| 英雄突破 | 是 |
| 技能升级 | 是 |
| 装备/卸下宝物 | 是 |
| 战斗结束 | 否 |

### 10.2 战力展示优化

```javascript
// 伪代码：战力格式化显示
function formatPower(power) {
  if (power >= 100000000) {
    return (power / 100000000).toFixed(1) + '亿';
  } else if (power >= 10000) {
    return (power / 10000).toFixed(1) + '万';
  } else {
    return power.toString();
  }
}
```

### 10.3 战力同步策略

1. **实时同步**：英雄属性变化时立即更新战力
2. **异步更新**：宝物战力变更可延迟计算
3. **缓存策略**：读取时使用缓存，变更时更新
