## 一、系统概述

### 1.1 设计理念

**核心定位**：探险队系统是本游戏的核心挂机玩法，玩家派出英雄队伍执行探险任务，离线也能获得收益。

**核心体验**：
- 系统刷新探险任务，玩家选择执行
- 任务难度随主线进度提升
- 英雄类型和战力影响任务结果
- 离线继续执行，上线收菜

**核心约束**：
- 派出探险的英雄不能同时参与主线关卡
- 玩家需要平衡：派谁去探险（收益）、留谁在队伍（战斗）
- 高战力英雄的资源分配策略

### 1.2 与主线进度的关系

```
主线关卡进度 → 解锁探险任务类型 → 影响任务难度等级

示例：
├── 通过第1-10关 → 解锁初级探险任务
├── 通过第11-20关 → 解锁中级探险任务
├── 通过第21-30关 → 解锁高级探险任务
└── 通过第31-40关 → 解锁精英探险任务
```

## 四、英雄占用机制

### 4.1 核心规则

**派出探险的英雄不能同时参与主线关卡**

```
┌─────────────────────────────────────────────────────────────┐
│                    英雄占用规则                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  英雄状态：                                                  │
│  ├── 空闲：可参与主线关卡、可派去探险                       │
│  ├── 探险中：不能参与主线关卡，需等待探险结束               │
│  └── 主线中：不能派去探险（正在随玩家下墓）                 │
│                                                             │
│  冲突处理：                                                  │
│  ├── 派出探险时，自动从主线队伍中移除                       │
│  ├── 探险中的英雄显示"探险中"状态，不可选择                │
│  └── 探险结束后，英雄恢复空闲状态                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 策略决策点

**玩家需要平衡的问题**：

| 决策 | 考量因素 |
|------|---------|
| 派谁去探险？ | 任务需求、英雄类型加成、战力 |
| 留谁在主线？ | 主线难度、BOSS战需求、队伍配合 |
| 高战力英雄 | 派出去收益高？还是留着打主线？ |
| 低战力英雄 | 派出去成功率低？还是主线打不动？ |

**典型策略**：
- **主力留主线**：高战力英雄留着打困难关卡，低战力派去低难度探险
- **收益优先**：高战力英雄派去高难度探险获得稀有奖励，用低战力勉强推图
- **平衡分配**：每队都有核心+辅助，兼顾主线和探险

### 4.4 召回机制

**玩家可以主动召回探险中的英雄**：
- 召回后英雄立即恢复空闲状态
- 但探险任务中断，进度作废，无奖励
- 适用场景：突然需要英雄打主线BOSS/竞技场

## 五、任务成功率计算

### 5.1 成功率公式

```
最终成功率 = 基础成功率 + 战力加成 + 阵容加成 + 英雄类型加成

各部分说明：

1. 基础成功率（根据难度）
   - 初级：80%
   - 中级：70%
   - 高级：60%
   - 精英：50%

2. 战力加成
   战力加成 = (队伍战力 - 需求战力) / 需求战力 × 20%
   上限：+20%
   下限：-30%

3. 阵容加成（英雄数量）
   - 满编3人：+10%
   - 2人：+5%
   - 1人：+0%

4. 英雄类型加成（见下表）
```

### 5.2 英雄类型加成表

| 任务类型 | 推荐英雄类型 | 加成 |
|----------|--------------|:----:|
| 资源探索 | 任意 | +0% |
| 碎片搜寻 | 对应传承英雄 | +15% |
| 宝物鉴定 | 摸金手（探测型） | +15% |
| 守护挑战 | 铁柱（坦克型） | +15% |
| 传承试炼 | 对应传承英雄 | +20% |
| 星核寻觅 | 四大传承各一人 | +25% |
| 精英远征 | SSR英雄 | +15% |

### 4.3 成功率示例

```
任务：星核寻觅（高级）
- 基础成功率：60%
- 需求战力：65,000
- 推荐阵容：四大传承各一人

玩家派出：
- 队伍战力：70,000
- 阵容：陈星垣（星纹摸金）+ 云绮罗（璇玑搬山）+ 铁柱（天枢发丘）

计算：
1. 基础成功率：60%
2. 战力加成：(70000-65000)/65000 × 20% = +1.5%
3. 阵容加成：+10%（满编）
4. 英雄类型加成：+15%（有三个传承，缺一个）

最终成功率 = 60% + 1.5% + 10% + 15% = 86.5%
```
