## 二、英雄升级经验曲线

### 2.1 经验曲线公式

采用**指数增长 + 分段调整**模式：

```
ExpRequired(level) = BaseExp × (level ^ GrowthExponent) × SegmentMultiplier(level)
```

**简化版本（实际使用）：**

```
ExpRequired(level) = BaseExp × level × (1 + level × 0.01) × SegmentFactor
```

### 2.2 分段系数表

| 等级区间 | 段系数 | 说明 |
|----------|--------|------|
| 1-30 | 1.0 | 新手期，曲线平缓 |

| 121-180 | 2.0 | 后期，高速增长 |
| 181-250 | 2.8 | 进阶期，指数增长 |
| 251-300 | 4.0 | 巅峰期，极速增长 |

## 十、经验系统使用指南

### 10.1 等级经验计算

```javascript
// 伪代码：计算升级所需经验
function calculateExpToLevel(currentLevel, targetLevel) {
  let totalExp = 0;
  for (let level = currentLevel + 1; level <= targetLevel; level++) {
    totalExp += getExpRequired(level);
  }
  return totalExp;
}

// 获取单级经验
function getExpRequired(level) {
  const segmentFactor = getSegmentFactor(level);
  const baseExp = 100;
  return Math.floor(baseExp * level * (1 + level * 0.015) * segmentFactor);
}
```

### 10.2 等级经验预估

```javascript
// 伪代码：根据经验估算等级
function estimateLevel(totalExp) {
  let level = 1;
  let accumulatedExp = 0;
  
  while (accumulatedExp < totalExp && level < 300) {
    accumulatedExp += getExpRequired(level);
    level++;
  }
  
  return level - 1;
}
```
