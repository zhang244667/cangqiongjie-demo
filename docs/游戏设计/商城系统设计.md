## 一、系统概述

### 1.1 系统定位

商城系统是《苍穹劫·摸金传人》的核心商业化入口，为玩家提供资源购买、礼包订阅、限时特惠等服务。系统设计遵循"免费可玩、付费加速"原则，确保免费玩家正常游戏体验的同时，为付费玩家提供便捷的资源获取渠道。

### 1.2 设计目标

| 目标 | 说明 |
|------|------|
| **资源供给** | 为玩家提供便捷的资源获取渠道，解决养成瓶颈 |
| **付费转化** | 通过超值礼包、限时折扣引导免费玩家首次付费 |
| **价值感知** | 通过原价对比、折扣展示，强化付费价值感 |
| **长线留存** | 通过月卡、订阅等持续性付费提升留存 |

### 1.3 设计原则

- **非强制**：商城非游戏必需入口，免费玩家可正常游玩
- **价值透明**：价格、内容、折扣清晰可见
- **分层供给**：小额、中额、大额商品分层满足不同玩家
- **限时刺激**：通过限时特惠制造紧迫感

#### 3.2.3 订阅商品（月卡）

**定义**：周期性付费，持续获得收益的订阅服务

| 月卡类型 | 价格 | 即时奖励 | 每日奖励 | 持续时间 |
|---------|------|---------|---------|---------|
| 探险月卡 | ¥30 | 💎300 | 💎150 + ⚡×60 | 30天 |
| 至尊月卡 | ¥68 | 💎680 | 💎300 + ⚡×120 + 🪙×10000 | 30天 |
| 豪华月卡 | ¥128 | 💎1280 | 💎500 + ⚡×180 + 🟢召唤券×1 | 30天 |

- 购买立即获得首笔奖励
- 每日登录领取每日奖励
- 未登录不补发，但到期后可补领未领取的天数（最多补领3天）
- 可叠加购买，时间累加

#### 3.2.4 限时特惠

**定义**：限时限量，折扣力度更大的促销商品

| 特惠类型 | 触发条件 | 示例商品 | 折扣力度 |

## 四、特色功能设计

### 4.1 智能推荐系统

根据玩家行为数据，推荐最合适的商品：

| 推荐场景 | 推荐商品 | 触发条件 |
|---------|---------|---------|
| 资源不足 | 金币/体力礼包 | 金币<1000 或 体力<30 |
| 养成卡点 | 强化材料包 | 英雄强化材料不足 |
| 新手引导 | 新手礼包 | 账号等级<10 且 未购买 |
| 首充引导 | 首充礼包 | 未充值 且 游戏时长>1小时 |
| 回归玩家 | 回归礼包 | 离线>7天 |

### 4.2 超值日历

每日一款超值商品，刺激日活：

| 星期 | 特惠商品 | 折扣 |
|------|---------|------|
| 周一 | 体力特惠 | 8折 |
| 周二 | 金币特惠 | 8折 |
| 周三 | 材料特惠 | 8折 |
| 周四 | 召唤券特惠 | 7折 |
| 周五 | 经验特惠 | 8折 |
| 周六 | 周末大礼包 | 6折 |
| 周日 | 周末大礼包 | 6折 |

### 4.3 首充双倍

**规则**：
- 首次充值任意金额，获得双倍钻石
- 仅限第一次充值有效
- 显示"首充双倍"标签

**示例**：
```
原价：¥6 = 60钻石
首充：¥6 = 60 + 60 = 120钻石
```

### 4.4 礼包码兑换

**功能**：支持输入礼包码兑换奖励

**入口**：商城右上角"兑换"按钮

**流程**：
```
点击兑换 → 输入礼包码 → 验证 → 发放奖励 → 显示结果
```

## 七、数据埋点设计

### 7.1 关键事件

| 事件名 | 触发时机 | 参数 |
|--------|---------|------|
| shop_view | 进入商城 | tab_name |
| item_click | 点击商品 | item_id, item_name |
| item_buy | 购买商品 | item_id, item_name, price, count |
| pay_start | 发起支付 | item_id, amount |
| pay_success | 支付成功 | item_id, amount, order_id |
| pay_fail | 支付失败 | item_id, amount, error_code |

### 7.2 关键指标

| 指标 | 计算方式 | 目标值 |
|------|---------|--------|
| 商城渗透率 | 进入商城人数 / 日活 | >30% |
| 商品点击率 | 商品点击次数 / 商品展示次数 | >10% |
| 购买转化率 | 购买人数 / 点击人数 | >5% |
| 客单价 | 总收入 / 购买人数 | >50元 |
| 首充转化率 | 首充人数 / 注册人数 | >10% |
