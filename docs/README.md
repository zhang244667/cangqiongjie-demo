# 《苍穹劫》IP企划总览

> 盗墓探险 × 星空占星 × 上古神话 × 宿命诅咒

---

## 项目简介

《苍穹劫》是一个原创IP企划，核心概念：**盗墓不是掘墓取宝，而是修补破碎苍穹**。

古墓即星盘，棺椁即星位，机关即星轨。玩家扮演传承者，探索古墓、破解机关、收集星核，最终重补天穹。

**系列规划**：
- 第一部《苍穹劫·摸金传人》（当前开发）
- 第二部《苍穹劫·紫微真相》
- 第三部《苍穹劫·星陨录》（前传）
