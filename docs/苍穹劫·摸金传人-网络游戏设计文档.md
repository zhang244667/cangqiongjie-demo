## 一、游戏定位与核心玩法

### 1.1 游戏类型
- **主类型**：挂机放置角色养成手机游戏
- **核心玩法**：关卡内策略探索 + 关卡外放置养成
- **支持平台**：手机APP（iOS/Android）优先，未来兼容微信小游戏
- **开发引擎**：Cocos Creator

### 1.2 核心设计理念
- 关卡内不考虑放置特性，纯策略探索玩法
- 探索方式：手动翻格子
- 解谜深度：轻度，点击触发为主
- 战斗与解谜比例：70%战斗 + 30%解谜
- 关卡时长：普通关卡10分钟，挑战/活动关卡30分钟+

### 1.3 立绘风格偏好
- 倾向于《闪烁之光》那种不带背景的透明底立绘
- 可通过image_generate工具的抠图模式制作透明底立绘

## 二、剧情框架与拓展体系

### 2.1 三部曲规划

#### 游戏一：《苍穹劫·四象篇》（当前开发）
- **定位**：入门篇、世界观建立、传统盗墓冒险
- **剧情**：第1-3层真相，陈星垣寻爷爷下落，结识四大传承伙伴，收集四象星核以为"补天"，遭遇归墟组织追杀
- **结局**：获得紫微玄穹墓线索，TO BE CONTINUED

#### 游戏二：《苍穹劫·紫微篇》
- **定位**：真相篇、叙事高潮、道德抉择
- **剧情**：第4-5层真相，发现"补天是谎言"，星核是封印碎片，主角是"钥匙"，面临终极三结局抉择

#### 游戏三：《苍穹劫·星陨篇》（前传）
- **定位**：起源篇、世界观补完、史诗传奇
- **剧情**：亿万年前苍穹劫爆发，上古星祭师与初代盗墓先贤联手封印星核，创建四大传承体系

### 2.2 《四象篇》完整故事结构（起承转合）
- **起**：爷爷失踪，星垣首次下墓觉醒星纹，结识铁柱
- **承**：四象古墓探险，依次结识星祭司、摸金手、搬山客，收集星核，逐步了解"补天"使命
- **转**：爷爷影像警告，归墟首领暗示阴谋，守墓神族阻止前往紫微墓
- **合**：启程昆仑，准备揭开终极真相

### 2.3 拓展框架设计

#### 主线分期发布策略
把"四象篇"拆成4个大版本，每个版本一个古墓：

```
【1.0 青龙篇】
    ├─ 青龙古墓（主线）
    ├─ 青龙支线：龙脉之源
    └─ 角色：铁柱加入
    
【1.5 白虎篇】
    ├─ 白虎古墓（主线）
    ├─ 白虎支线：凶煞之谜
    └─ 角色：摸金手加入
    
【2.0 朱雀篇】
    ├─ 朱雀古墓（主线）
    ├─ 朱雀支线：涅槃传说
    └─ 角色：搬山客加入
    
【2.5 玄武篇】
    ├─ 玄武古墓（主线）
    ├─ 玄武支线：万年守护
    └─ 四象合璧 → 解锁紫微线索
```

**更新节奏**：每2-3个月一个大版本

#### 支线剧情系统（无限拓展空间）

**1. 角色故事线（个人剧情）**
每个角色都有独立的剧情线，不与主线强绑定：

| 角色 | 个人剧情主题 | 更新方式 |
|------|-------------|----------|
| 铁柱 | 发丘印的诅咒 | 每2周1章，共6章 |
| 星祭司 | 星纹的秘密 | 每2周1章，共6章 |
| 摸金手 | 失踪的族人 | 每2周1章，共6章 |
| 搬山客 | 机关术传承 | 每2周1章，共6章 |
| **新角色** | 各自背景故事 | 新角色上线时开启 |

**拓展逻辑**：每出一个新角色，就自带一条个人剧情线

**2. 四大传承深度剧情**
深挖每个传承的历史和秘密：

```
【星纹摸金传】
    ├─ 第1章：初代摸金
    ├─ 第2章：星纹的代价
    ├─ 第3章：三十岁诅咒的起源
    ├─ ...
    └─ 第6章：爷爷的去向（暗示紫微篇）
```

**3. 势力剧情线（第三方视角）**

```
【归墟档案】（从反派视角理解故事）
    ├─ 归墟首领的过去
    ├─ 为什么需要"钥匙"
    ├─ 归墟成员的个人故事
    └─ 归墟的真正目的（紫微篇揭晓）

【守墓神族志】
    ├─ 守墓的使命
    ├─ 与盗墓者的千年恩怨
    ├─ 星噬现象的真相
    └─ 神族内部的分歧

【科考队纪实】
    ├─ 官方调查视角
    ├─ 苏博士的研究笔记
    └─ 科学 vs 神秘的碰撞
```

#### 平行古墓系统（独立于主线）

在四象古墓之外，设计大量"平行古墓"：

| 类型 | 特点 | 更新方式 |
|------|------|----------|
| **传承古墓** | 某个传承的专属古墓，掉落传承装备 | 随新传承剧情更新 |
| **星陨古墓** | 陨星坠落形成的古墓，高难度副本 | 每月1个 |
| **活动古墓** | 节日限定，限时开放 | 节日活动 |
| **深渊古墓** | 无限层级，爬塔玩法 | 常驻，定期加层 |
| **随机古墓** | 程序生成，每次不同 | 常驻 |

#### 赛季制设计

用"赛季"包装剧情更新，给玩家持续的新鲜感：

```
【第1赛季：青龙觉醒】
    ├─ 青龙古墓主线
    ├─ 赛季专属角色/装备
    └─ 赛季排名奖励

【第2赛季：白虎凶煞】
    ├─ 白虎古墓主线
    ├─ 归墟组织入侵（赛季事件）
    └─ 新角色：摸金手

【第3赛季：朱雀涅槃】
    ├─ 朱雀古墓主线
    ├─ 守墓神族觉醒（赛季事件）
    └─ 新角色：搬山客

【第4赛季：玄武千年】
    ├─ 玄武古墓主线
    ├─ 科考队介入（赛季事件）
    └─ 四象合璧活动
```

#### 世界观预留拓展点

**1. 未开放的地图区域**
```
【已开放】青龙、白虎、朱雀、玄武四墓
【未开放】
    ├─ 南斗六星墓（预留）
    ├─ 北斗七星墓（预留）
    ├─ 二十八宿墓群（预留）
    └─ 紫微玄穹墓（紫微篇开放）
```

**2. 未揭示的势力**
```
【已知势力】
    ├─ 星脉传人
    ├─ 守墓神族
    ├─ 官方科考队
    └─ 归墟组织
    
【预留势力】
    ├─ 星宫后裔（上古星宫的血脉）
    ├─ 天外访客（苍穹劫的真正来源）
    └─ 其他...（根据运营需要拓展）
```

## 三、单篇章内容量设计

### 3.1 内容量计算基准

假设一个普通玩家：
- 每天游戏30分钟
- 每周约3.5小时
- 一个版本周期2-3个月 = 约30-40小时游戏时间

**单个篇章需要填充30-40小时的内容**

### 3.2 青龙篇详细内容拆解

#### 一、主线剧情（约8小时）

| 章节 | 内容 | 时长 |
|------|------|------|
| 第1章 | 爷爷失踪，初入青龙墓 | 1小时 |
| 第2章 | 探索青龙墓外围，遭遇尸傀 | 1.5小时 |
| 第3章 | 深入青龙墓核心，解谜 | 2小时 |
| 第4章 | 青龙守墓灵Boss战 | 1小时 |
| 第5章 | 获得青龙星核，铁柱加入 | 1小时 |
| 第6章 | 归墟初现，首次交锋 | 1.5小时 |

**主线特点**：一次性体验，看剧情为主

#### 二、角色剧情（约6小时）

| 角色 | 章节数 | 总时长 |
|------|--------|--------|
| 陈星垣个人剧情 | 6章 | 2小时 |
| 铁柱个人剧情 | 6章 | 2小时 |
| 星祭司个人剧情 | 6章 | 2小时 |

**更新方式**：每周解锁1章，持续6周

#### 三、探索内容（约10小时）

**青龙古墓探索**：

| 内容 | 层数/关卡数 | 时长 |
|------|-------------|------|
| 青龙墓外围 | 10层 | 2小时 |
| 青龙墓中层 | 15层 | 3小时 |
| 青龙墓核心 | 10层 | 2小时 |
| 青龙隐藏区域 | 5层（解谜解锁） | 1.5小时 |
| 青龙深渊 | 无限层 | 持续 |

**探索特点**：可重复刷，有随机性

#### 四、副本内容（约8小时）

| 副本类型 | 数量 | 时长 |
|----------|------|------|
| 青龙守墓灵（普通） | 1个 | 30分钟 |
| 青龙守墓灵（困难） | 1个 | 30分钟 |
| 青龙守墓灵（噩梦） | 1个 | 30分钟 |
| 归墟精英副本 | 3个 | 各30分钟 |
| 青龙传承试炼 | 1个 | 1小时 |
| 青龙深渊Boss | 每周刷新 | 持续 |

**副本特点**：需要反复刷装备/材料

#### 五、养成内容（持续消耗时间）

| 系统 | 每日耗时 | 周期 |
|------|----------|------|
| 日常任务 | 10分钟 | 每日 |
| 角色升级 | 10分钟 | 每日 |
| 装备强化 | 5分钟 | 每日 |
| 传承培养 | 5分钟 | 每日 |
| 公会活动 | 15分钟 | 每日 |

**养成特点**：每天约45分钟，2个月约45小时

#### 六、活动内容（版本周期内）

| 活动 | 时长 | 频率 |
|------|------|------|
| 青龙祭典（版本活动） | 2周 | 版本开启时 |
| 星噬危机（限时活动） | 3天 | 每月1次 |
| 归墟入侵（世界Boss） | 1天 | 每周1次 |
| 青龙传承赛（竞技） | 1周 | 版本中期 |

#### 青龙篇总内容量汇总

| 内容类型 | 首通时长 | 可重复时长 |
|----------|----------|------------|
| 主线剧情 | 8小时 | 不可重复 |
| 角色剧情 | 6小时 | 不可重复 |
| 探索关卡 | 10小时 | 可反复刷 |
| 副本挑战 | 8小时 | 可反复刷 |
| 养成系统 | - | 45小时/2月 |
| 活动内容 | 10小时 | 限时 |

**首通总时长**：约40小时
**2个月总时长**：约80小时（含重复内容）

### 3.3 四篇内容对比

| 篇章 | 主线时长 | 特色内容 | 核心差异点 |
|------|----------|----------|------------|
| **青龙篇** | 8小时 | 探索教学、角色加入 | 入门篇，建立基础 |
| **白虎篇** | 10小时 | 归墟追杀、PVP竞技 | 冲突升级，引入对抗 |
| **朱雀篇** | 12小时 | 三方势力、公会战 | 势力碰撞，社交深化 |
| **玄武篇** | 15小时 | 高难副本、真相初露 | 难度巅峰，剧情高潮 |

### 3.4 版本更新节奏细化

以青龙篇为例，2个月的更新计划：

| 周次 | 更新内容 |
|------|----------|
| 第1周 | 版本上线：主线第1-2章 + 探索10层 |
| 第2周 | 主线第3章 + 角色剧情解锁 |
| 第3周 | 主线第4章 + 归墟副本 |
| 第4周 | 主线第5-6章（完结）+ 青龙祭典活动 |
| 第5周 | 青龙传承试炼 + 深渊模式 |
| 第6周 | 角色剧情完结 + 困难副本 |
| 第7周 | 噩梦副本 + 限时活动 |
| 第8周 | 青龙篇收尾活动 + 白虎篇预告 |

### 3.5 玩家视角体验曲线

```
【第1-2周】热情期
    └─ 主线推进，新鲜感强，每天玩1小时

【第3-4周】深入期
    └─ 主线完结，开始刷探索/副本，每天45分钟

【第5-6周】平稳期
    └─ 养成为主，角色剧情，每天30分钟

【第7-8周】期待期
    └─ 活动收尾，等新版本，每天20分钟
```

## 四、角色系统设计

### 4.1 初始角色阵容（30人）

#### 分配原则
- 四大传承：每传承5人 = 20人
- 守墓神族：4人
- 归墟组织：4人（可玩）
- 其他势力：2人

- 背景：星纹摸金资深传人，陈老爷子的老友

**阿蛮**
- 性别：女
- 外貌：精瘦敏捷，双马尾，眼神灵动
- 武器：双匕首
- 性格：活泼好动，嘴硬心软

**星语**

- 性格：温柔可靠，姐姐型
- 背景：陈星垣的表姐，也是星纹摸金传人，擅长医疗和辅助

### 5.2 各势力详细设计

#### 【星纹摸金】

**历史渊源**：
- 源自三国曹操设立的"摸金校尉"
- 传承1800年，秘籍《摸金定穴经》
- 历代传人皆为风水大师
- 清末民国时期逐渐转为民间组织

**核心理念**：星纹引路，摸金探秘
**真实文化连接**：
- 风水学：罗盘定位、寻龙点穴、望气之法
- 易经八卦：乾坎艮震巽离坤兑
- 阴阳五行：金木水火土相生相克

**特色**：星象定位、探测机关、速度型

---

#### 【天枢发丘】

**历史渊源**：
- 与摸金校尉同时设立，"发丘中郎将"
- 专司开墓破阵，手持"发丘印"
- 明代后转为民间门派
- 以力量和机关术见长

**核心理念**：发丘印开路，守护同门
**真实文化连接**：
- 发丘印：传说中能号令鬼神的印信
- 机关术：墨家机关术的传承分支
- 道教符咒：开山破墓的符法

**特色**：高防御、力量型、团队保护

---

#### 【璇玑搬山】

**历史渊源**：
- 源自古代西域（今新疆、中亚）
- 传承"穿山甲"秘术
- 历史上与丝绸之路商队有密切联系
- 以奇门遁甲、丹药毒术见长

**核心理念**：星术探秘，医毒双修
**真实文化连接**：
- 西域文化：龟兹、楼兰、精绝等古国
- 奇门遁甲：九宫八卦、天干地支
- 丹药术：炼丹术、医毒双修

**特色**：法术伤害、治疗辅助、星术

---

#### 【辰宿卸岭】
