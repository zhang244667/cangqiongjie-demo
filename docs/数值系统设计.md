## 一、设计总纲

### 1.1 核心设计目标

| 设计维度 | 目标定位 | 设计理念 |
|---------|---------|---------|
| **稀有度差距** | 小差距 | SSR比SR强30%，UR比SSR强15%，让SR角色保持长期价值 |
| **克制关系** | 三层深入 | 传承克制+属性克制+敌人类型克制，策略大于数值 |
| **养成节奏** | 中等 | 1个月1个5星，3个月3个6星，稳定可预期 |
| **PVP定位** | 辅助玩法 | 全自动战斗，轻度禁用，中等付费获取限定角色 |

### 1.2 数值设计公式

**伤害计算基础公式**：
```
最终伤害 = (攻击力 × 技能倍率 - 防御力 × 减伤系数) × 克制系数 × 暴击系数 × 随机波动
```

**属性成长公式**：
```
角色属性 = 基础值 × (1 + 等级系数) × 星级系数 × 装备系数 × 技能系数
```

## 三、克制关系系统

### 3.1 三层克制架构

```
┌─────────────────────────────────────┐
│        第一层：传承克制              │
│   (四大传承间的循环克制)             │
├─────────────────────────────────────┤
│        第二层：属性克制              │
│   (五行属性间的相生相克)             │
├─────────────────────────────────────┤
│        第三层：敌人类型克制          │
│   (角色定位vs敌人类型)               │
└─────────────────────────────────────┘
```

### 3.2 第一层：传承克制

**四大传承循环克制**：
```
星纹摸金 → 克制 → 璇玑搬山
    ↓                  ↑
  被克                被克
    ↑                  ↓
辰宿卸岭 ← 克制 ← 天枢发丘
```

**克制效果**：
| 关系 | 伤害加成 | 受伤减免 |
|------|---------|---------|
| 克制对方 | +30% | - |
| 被对方克制 | - | +20% |
| 无克制关系 | - | - |

**最大叠加效果**：1.56倍（克制方打被克方）

**传承克制背景逻辑**：

| 传承 | 克制原理 | 战斗体现 |
|------|---------|---------|
| 星纹摸金 → 璇玑搬山 | 星术破解机关预判 | 技能命中+15%，对机关敌人额外伤害 |
| 璇玑搬山 → 天枢发丘 | 机关克制重装突破 | 暴击率+10%，对重装敌人穿透+20% |
| 天枢发丘 → 辰宿卸岭 | 重装压制技术流 | 击退效果+30%，对技术敌人控制时间+20% |
| 辰宿卸岭 → 星纹摸金 | 技术克制星术施法 | 技能冷却-10%，对施法敌人打断概率+25% |

### 3.3 第二层：属性克制

**五行相克循环**：
```
金 → 克制 → 木
↑           ↓
土         火
↑           ↓
水 ← 克制 ← 火
```

**属性克制效果**：

| 克制关系 | 伤害加成 | 视觉效果 |
|---------|---------|---------|
| 金克木 | +20% | 金色光芒 |
| 木克土 | +20% | 绿色藤蔓 |
| 土克水 | +20% | 黄色岩石 |
| 水克火 | +20% | 蓝色水流 |
| 火克金 | +20% | 红色火焰 |

**属性相生（辅助效果）**：

| 相生关系 | 效果 |
|---------|------|
| 金生水 | 水属性角色获得+10%生命上限 |
| 木生火 | 火属性角色获得+10%暴击伤害 |
| 水生木 | 木属性角色获得+10%治疗效果 |
| 火生土 | 土属性角色获得+10%防御力 |
| 土生金 | 金属性角色获得+10%攻击力 |

**队伍属性共鸣**：
- 同属性2人：+5%对应属性效果
- 同属性3人：+12%对应属性效果
- 同属性4人：+20%对应属性效果
- 同属性5人：+30%对应属性效果+专属特效

### 3.4 第三层：敌人类型克制

**敌人类型分类**：

| 类型 | 特点 | 弱点 | 克制角色 |
|------|------|------|---------|
| **重装型** | 高防高血，慢速 | 真实伤害、穿透 | 法师、刺客 |
| **敏捷型** | 高闪避高暴击，低血 | 控制技能、必中 | 控制型、坦克 |
| **施法型** | 高技能伤害，施法前摇 | 打断、沉默 | 战士、刺客 |
| **召唤型** | 召唤小怪，本体脆弱 | 范围伤害 | AOE角色 |
| **机关型** | 技能复杂，有规律 | 机关破解 | 搬山传承 |
| **不死型** | 复活、吸血 | 神圣伤害 | 守墓神族 |

**克制效果表**：

| 角色定位 | 克制敌人类型 | 额外效果 |
|---------|-------------|---------|
| 战士 | 施法型 | 打断概率+25% |
| 法师 | 重装型 | 穿透+30% |
| 刺客 | 施法型、敏捷型 | 暴击率+15% |
| 坦克 | 敏捷型 | 反伤+20% |
| 辅助 | 召唤型 | 召唤物伤害-30% |
| 机关师 | 机关型 | 机关破解速度+50% |

### 3.5 克制叠加计算

**叠加规则**：
1. 三层克制独立计算
2. 伤害加成乘法叠加
3. 减伤效果加法叠加

**示例计算**：
```
场景：星纹摸金的火属性法师攻击璇玑搬山的木属性重装敌人

第一层传承克制：星纹摸金 → 璇玑搬山 = +30%伤害
第二层属性克制：火克木 = +20%伤害
第三层类型克制：法师 → 重装型 = +30%穿透

最终伤害倍率 = 1.3 × 1.2 × 1.3(穿透等效) = 2.03倍
```

**最大克制倍率上限**：2.5倍（防止数值膨胀）

## 五、PVP竞技系统

### 5.1 PVP定位与设计理念

**PVP定位**：辅助玩法，不强制参与

| 设计维度 | 具体策略 |
|---------|---------|
| **参与门槛** | 低，30级解锁，无强制要求 |
| **奖励导向** | 外观奖励为主，数值奖励为辅 |
| **时间投入** | 低，每日3-5场即可获取主要奖励 |
| **付费优势** | 有，但控制在30%以内 |

### 5.2 战斗机制

**全自动战斗规则**：

| 规则项 | 设计 |
|--------|------|
| **战斗模式** | 5v5回合制，全自动 |
| **出手顺序** | 按速度值排序，同速随机 |
| **技能释放** | AI自动判断，优先高收益技能 |
| **战斗时长** | 最长3分钟，超时判定为攻击方失败 |
| **胜负判定** | 全灭或血量百分比 |

**AI战斗逻辑优先级**：

```
1. 控制技能优先打敌方输出
2. 治疗技能优先奶血量最低友方
3. 输出技能优先打克制目标
4. 防御技能在血量<30%时触发
5. 终极技能在最优时机释放（敌方聚集/残血收割）
```

### 5.3 赛季制度

**赛季周期**：6周一个赛季

| 赛季阶段 | 时长 | 内容 |
|---------|------|------|
| 预热期 | 3天 | 赛季任务开启，禁用名单公布 |
| 正式期 | 5周 | 排位赛，积分累计 |
| 结算期 | 3天 | 奖励发放，下赛季预告 |

**禁用规则（轻度）**：

| 禁用类型 | 数量 | 选择方式 |
|---------|------|---------|
| 强制禁用 | 1个 | 上赛季使用率最高的SSR |
| 投票禁用 | 1个 | 玩家投票选出 |
| 总计 | 1-2个 | 保持环境多样性 |

**禁用效果**：
- 禁用角色在排位赛中无法使用
- 禁用角色仍可在PVE中使用
- 禁用角色保有率补偿（免费碎片）

### 5.4 排位与奖励

**段位体系**：

| 段位 | 分数区间 | 人口比例 | 奖励 |
|------|---------|---------|------|
| 青铜 | 0-999 | 20% | 基础奖励 |
| 白银 | 1000-1999 | 30% | 基础+皮肤碎片 |
| 黄金 | 2000-2999 | 25% | 皮肤+SSR碎片 |
| 铂金 | 3000-3999 | 15% | 限定皮肤+SSR保底 |
| 钻石 | 4000-4999 | 7% | 专属皮肤+UR碎片 |
| 大师 | 5000-5999 | 2.5% | 专属皮肤+UR保底 |
| 宗师 | 6000+ | 0.5% | 全套专属+限定角色 |

**赛季奖励设计**：

| 奖励类型 | 获取条件 | 是否影响数值 |
|---------|---------|-------------|
| 段位皮肤 | 白银以上 | ❌ 纯外观 |
| 表情包 | 黄金以上 | ❌ 社交功能 |
| 称号 | 铂金以上 | ❌ 展示功能 |
| SSR碎片 | 黄金以上 | ✅ 养成加速 |
| UR碎片 | 大师以上 | ✅ 限定获取 |
| 限定角色 | 宗师 | ✅ 唯一获取途径 |

### 5.5 限定角色获取

**获取方式（中等付费）**：

| 获取途径 | 成本 | 时间要求 | 说明 |
|---------|------|---------|------|
| **赛季奖励** | 时间投入 | 宗师段位 | 免费玩家可达 |
| **赛季复刻** | 80-100抽 | 赛季后3个月 | 付费友好 |
| **赛季兑换** | 200碎片 | 赛季后6个月 | 长期积累兑换 |
| **免费获取** | - | 赛季后12个月 | 免费碎片兑换 |

**中等付费设计**：
- 月卡用户：3-6个月可复刻获取
- 中付费用户：赛季内可获取或快速复刻
- 免费玩家：需要1年时间，但可获取

**限定角色数值控制**：
- UR稀有度：比SSR强15%
- 不影响PVE核心体验
- PVP环境中可通过策略克制

### 5.6 匹配与公平性

**匹配机制**：

| 规则 | 说明 |
|------|------|
| 段位匹配 | ±1段位内匹配 |
| 战力参考 | 隐藏战力分，辅助匹配 |
| 胜率平衡 | 连败保护，匹配略弱对手 |
| 时间限制 | 匹配超时30秒，扩大范围 |

**公平性保障**：

| 保障项 | 措施 |
|--------|------|
| 数值差异 | 付费优势控制在30%以内 |
| 策略优先 | 克制系统放大策略价值 |
| 禁用机制 | 防止单一角色垄断 |
| 赛季重置 | 每6周环境洗牌 |

## 七、数值平衡工具

### 7.1 平衡监控指标

**角色强度监控**：

| 指标 | 计算方式 | 目标范围 |
|------|---------|---------|
| **使用率** | 选用次数/总场次 | 5-15%（健康） |
| **胜率** | 胜利场次/总场次 | 45-55%（健康） |
| **Ban率** | 禁用次数/总场次 | <10%（健康） |
| **伤害贡献** | 总伤害/团队伤害 | 按定位分布 |

**环境健康度指标**：

| 指标 | 说明 | 目标 |
|------|------|------|
| 角色多样性 | 不同角色使用数量 | >60%角色使用率>5% |
| 阵容多样性 | 不同阵容组合数量 | 无单一阵容>20% |
| 传承平衡 | 四大传承使用率 | 各传承差异<10% |
| 稀有度分布 | 各稀有度使用率 | SR占比>30% |

### 7.2 平衡调整机制

**定期调整**：

| 调整类型 | 频率 | 内容 |
|---------|------|------|
| 热修复 | 即时 | 严重Bug、数值错误 |
| 小更新 | 每2周 | 微调±5% |
| 中更新 | 每月 | 调整±10%，新增内容 |
| 大更新 | 每赛季 | 重做、新机制 |

**调整优先级**：

```
优先级1：影响游戏公平性的严重问题
优先级2：胜率>60%或<40%的角色
优先级3：使用率<2%的角色（加强）
优先级4：Ban率>15%的角色（削弱）
优先级5：环境单一化问题
```

### 7.3 测试验证流程

**数值测试流程**：

```
1. 理论计算 → 公式验证
     ↓
2. 模拟测试 → AI对战验证
     ↓
3. 小范围测试 → 玩家体验测试
     ↓
4. 数据监控 → 上线后数据追踪
     ↓
5. 反馈调整 → 根据实际数据调整
```

**测试案例设计**：

| 测试场景 | 测试内容 | 验证指标 |
|---------|---------|---------|
| 极端克制 | 最大克制叠加 | 伤害不超过2.5倍上限 |
| 稀有度差距 | SR vs SSR | SSR优势不超过30% |
| 付费差距 | 免费 vs 月卡 | 月卡优势不超过20% |
| 养成进度 | 新手 vs 老玩家 | 可通过策略弥补差距 |
