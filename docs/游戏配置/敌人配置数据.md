## 一、敌人类型概述

### 类型分布总览
| 类型 | 数量 | 说明 | 难度梯度 |
|:----:|:----:|------|:--------:|
| 小怪 | 20 | 普通敌人，造型多样 | 1-5 |
| 精英 | 10 | 强化版，特殊机制 | 4-7 |
| 主线Boss | 8 | 每章1个 | 6-10 |
| 四象守护 | 4 | 四象古墓Boss | 8-10 |
| 机关陷阱 | 8 | 非生物敌人 | 1-8 |

### 难度等级说明
- 等级1-3：新手期敌人，适合第1-2章
- 等级4-5：成长期敌人，适合第3-4章
- 等级6-7：中期敌人，适合第5-6章
- 等级8-9：后期敌人，适合第7-8章
- 等级10：终极敌人，四象守护和最终Boss

## 二、盗墓题材敌人风格

### 2.1 敌人题材分类
| 题材 | 典型代表 | 特点 |
|------|----------|------|
| 僵尸/粽子 | 跳尸、立僵 | 近战为主，攻击附带尸毒 |
| 墓俑/兵马俑 | 秦俑、陶俑 | 防御高，群体行动 |
| 机关兽 | 铁兽、铜牛 | 机械伤害，AOE攻击 |
| 守墓人 | 守墓将军、墓守 | 技能多样，各有特色 |
| 鬼魂/怨灵 | 游魂、厉鬼 | 隐身、穿墙、附身 |
| 骷髅兵 | 白骨战士、骷髅弓手 | 复活、范围攻击 |
| 石像/石狮 | 守门石狮、石人 | 坚固不移，反击伤害 |
| 墓虫/尸蹩 | 尸蹩群、墓蛾 | 群体攻击，中毒效果 |
| 幻象/鬼打墙 | 迷障、幻影 | 制造幻觉，分散注意力 |

### 2.2 敌人属性计算公式
```
最终属性 = 基础属性 × (1 + 难度系数) × (1 + 精英/Boss加成)
```
- 精英敌人：属性 × 1.5
- Boss敌人：属性 × 3.0
- 四象守护：属性 × 4.0

## 九、Boss特殊机制详解

### 9.1 Boss机制设计原则
1. **多样性**：每个Boss拥有独特的战斗机制
2. **策略性**：机制需要玩家调整阵容和战术
3. **挑战性**：机制增加战斗难度但不过于碾压
4. **趣味性**：机制带来独特的战斗体验

### 9.2 机制类型分类

#### 阶段型机制
- **多阶段Boss**：Boss在HP降至特定百分比时触发阶段变化
- **示例**：守墓鬼将、青龙守护

#### 召唤型机制
- **召唤小怪**：Boss定期召唤小怪助战
- **示例**：将军俑、冥将

#### 护盾型机制
- **护盾机制**：Boss拥有可刷新或累积的护盾
- **示例**：玄武封印、青龙守护

#### 复活型机制
- **死亡复活**：Boss拥有复活能力
- **示例**：朱雀炎魔

#### 状态型机制
- **狂暴状态**：Boss在特定条件下进入狂暴
- **示例**：千年古尸、守墓将军

### 9.3 应对策略指南

| Boss | 核心机制 | 推荐阵容 | 关键技巧 |
|------|----------|----------|----------|
| 守墓鬼将 | 多阶段变身 | 爆发输出 | 速战速决，避免拖入第三阶段 |
| 青龙守护 | 星轨护盾 | 控制+输出 | 先破护盾再输出 |
| 白虎战将 | 战意叠加 | 坦克+持续输出 | 需要足够坦度 |
| 朱雀炎魔 | 浴火重生 | 高爆发+驱散 | 预留技能击杀第二次 |
| 玄武封印 | 寒冰护甲+恢复 | 驱散+无视防御 | 带驱散技能 |
| 星噬魔 | 星噬光环 | 吸血+高爆发 | 速战速决 |
| 守墓将军 | 觉醒状态 | 免疫伤害应对+爆发 | 预留免疫技能 |
| 紫微星君 | 紫微命格 | 全方位高配阵容 | 需要极限战力和策略 |
